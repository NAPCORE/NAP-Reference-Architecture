This is the representation of the FRAME functionality that appears as the central "box"  in the Context Diagram.  Clicking on the System "box" will open Data Flow Diagram 0 (DFD) which is the highest level of Data Flow Diagram.  Alternatively click on the item below and this will take you to DFD0.<br/>DFD0 contains all of the Functional Areas, the data flows that link them together and the data flows that link the Functional Areas to the Terminators in the Context Diagram.  It provides the starting point for any exploration of the FRAME Functional View.<br/>
