It contains data objects that provide information about the different services available to the user and answers a request made by the user. The data flow may include the following data objects, depending on the request :<br/>- type of service (transport, information, parking, etc.),<br/>- type of information (schedule, access conditions, tariffs, etc.),<br/>- location of service,<br/>- date / time of service- tariff- operator ID.<br/>
