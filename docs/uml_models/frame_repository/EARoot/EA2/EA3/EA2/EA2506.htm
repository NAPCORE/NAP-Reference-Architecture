It contains data objects that provide instructions for the Freight Vehicle Driver about where to drop-off or collect the Freight Transport Load Item(s) to or from the other mode of transport.  The data objects shall include but not be limited to the following:<br/> - date,<br/> - time,<br/> - Freight Transport Execution Plan identity,<br/> - number of Freight Transport Load Items being moved through the implementation of the Plan,<br/> - expected time of arrival (eta),<br/> - location of the move onto the other mode of transport,<br/> - location of move back to road transport,<br/> - identity of required Freight Vehicle.<br/><br/>
