It contains outputs to the Pre-Trip Traveller that have been created from data objects provided by functionality in the Manage Public Transport Operations Functional Area and includes details of an On-Demand Service that has been created in response to a previous request from the Pre-Trip Traveller.  Details such as times of arrival and departure, route to be followed, use of other transport modes, type of Vehicle to be used and cost shall be included.<br/>
