Appears In: DFD F7. Provide Support for Law Enforcement
It contains outputs to the Law Enforcement Agency that have been created from data objects provided by functionality in the Provide Support for Law Enforcement Functional Area and includes the data objects that provide the response to a specific request from the Law Enforcement Agency for details of previous violations that have been recorded.  The data objects include some or all of the following:<br/>- reference,<br/>- date,<br/>- type of violation,<br/>- result of violation,<br/>- involved user ID (if available),<br/>- involved vehicle ID (if relevant and available),<br/>- image of violation (if available),<br/>- location of violation<br/>- any other data depending on the type of violation.<br/>
