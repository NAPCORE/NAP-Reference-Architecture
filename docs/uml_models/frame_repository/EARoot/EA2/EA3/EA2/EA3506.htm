It contains data objects that provide revised data about the way in which the Vehicle Trip Plan that a Driver is currently using is to be modified.  It is sent when it is necessary for the Trip Plan to be changed, either because the Driver has requested it, or because the road conditions have changed such that the trip plan needs to be improved.<br/>
