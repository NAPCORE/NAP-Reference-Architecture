It contains data objects that provide all the elements gathered by functionality in the Manage Traffic Functional Area on a violation affecting the use of a car park space. It includes the following data objects:<br/>- reference<br/>- date<br/>- length of time in excess of limit<br/>- image of violation (if available)<br/>- involved user ID (if available)<br/>- involved vehicle ID (if available)<br/>- car park identity and location<br/>
