It contains data objects about the Host Vehicle when it has initiated an e Call and will contain the following pieces of data:<br/>- the last two locations of the Host Vehicle<br/>- the level of confidence in the final Host Vehicle location data<br/>- the direction in which the Host Vehicle was travelling<br/>- method of eCall activation, i.e. manual or automatic (air bag)<br/>- Vehicle Identification Number (VIN) for the Host Vehicle<br/>- Number of passengers (how many seat belts fastened) in the Host Vehicle<br/>- Fuel type of the Host Vehicle, e.g. petrol, diesel, hybrid or electric<br/>- Is it a "test" eCall as part of Vehicle fitness checks?<br/>
