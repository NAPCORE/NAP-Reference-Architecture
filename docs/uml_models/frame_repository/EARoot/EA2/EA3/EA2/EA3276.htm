It contains data objects that identify an incorrect payment performed by a user and related to electronic payment.  The incorrect payment may either be an invalid payment, or an attempt to force one's way through the access control devices.  The data contains the following elements:<br/>- reference,<br/>- date,<br/>- type of incorrect payment,<br/>- result of incorrect payment,<br/>- involved user ID (if available),<br/>- involved vehicle ID (if relevant and available),<br/>- image of incorrect payment (if available),<br/>- location of at which incorrect payment was made,<br/>- other data according to the type of incorrect payment.<br/>
