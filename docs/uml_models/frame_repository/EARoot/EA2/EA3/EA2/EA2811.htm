It contains data objects that provide the contents of commands and information (c&i) messages that are for display and use by the Host Vehicle. The data objects will provide commands and messages for all types of Drivers to take actions so that the Host Vehicles will make best use of the urban road network. The location coded area where the command is to be displayed and where it is valid is also provided by the data objects.<br/>
