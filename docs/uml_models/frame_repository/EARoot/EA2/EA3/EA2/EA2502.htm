It contains data objects that provide details from which a new Freight Transport Execution Plan is to be prepared.  These data object comprise the following that have been provided by the Freight Transport User:<br/>- number of the Freight Transport Load Item(s), to be transported,<br/>- proposed itinerary for each Freight Transport Load Item comprising<br/>    - origin,<br/>    - expected departure date and time,<br/>    - destination,<br/>    - expected arrival date and time,<br/>- any constraints, e.g. Freight Transport Load Item(s) must be treated as an abnormal load<br/>In addition data objects containing any constraints added by the Freight Transport Operations Manager may be added to the data objects listed above.  THese data objects may comprise but not limited to:<br/>- choice of Freight Vehicle type to be used,<br/>- constraints on choice of route provided in the data from the Freight Transport User from which the Freight Transport Execution Plan was created,<br/> - mode(s) of transport that can be used in the implementation of the Freight Transport Execution Plan,<br/> - details of any traffic management polices that need to be followed when the Freight Transport Execution Plan is implemented..<br/>
