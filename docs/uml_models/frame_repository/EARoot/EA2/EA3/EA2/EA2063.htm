It contains inputs that a Driver has made which are used to create data objects that provide the record of vehicle data provided on-board the vehicle and liable to be checked by functionality in the Provide Support for Law Enforcement Functional Area.  The data objects that make up the record include details of the data such as how long the Driver has been driving the Vehicle and when the next rest stop is scheduled.<br/>
