It contains inputs the Emergency Vehicle Driver has made from which data objects are created to provide a request for virtual cones to be deployed at specified locations in the vicinity of an incident by functionality in the Provide Safety and Emergency Facilities Functional Area.  Virtual cones are used to define a part of the road network to which access is prohibited for ordinary vehicles (i.e. vehicles that do not belong to the Emergency Services) that have the capability to use in-vehicle systems for navigation and route following.  Thus the "cones" don't actually exist, except as in the information objects being used by the in-vehicle systems.<br/>
