It contains data objects that provide the notification that sensors in the Host Vehicle have detected that it has probably just been involved in an accident, because the Vehicle eCall Activation status has changed to "activated".  (Note: this does not need to involve other Vehicle(s)).  It also includes other data objects that provide the following:<br/>- the Vehicle e-call activation status,<br/>- Vehicle Identification Number (VIN),<br/>- the type of fuel used by the Host Vehicle<br/>- number of Host Vehicle occupants (the number of seat belts in use),<br/>- current position of the Host Vehicle,<br/>- previous position of the Host Vehicle,<br/>- Host Vehicle position confidence indicator,<br/>- direction Host Vehicle is travelling.<br/>
