It consists of inputs from the Driver and Vehicle Provider containing data objects that provide details about Vehicles that will enable the identities they provide through FCD/XFCD when passing through the inter-urban road network to be matched with their actual physical identities, i.e. is the FCD/XFCD credible for the type of Vehicle that it appears to be and is for use by functionality that compiles traffic data for the urban road network in the Manage Traffic Functional Area.  All data objects provided by this data flow must comply with the requirements of the General Data Protection Regulation (GDPR - EU2016/679 and any subsequent amendments.)<br/>
