It contains data objects that provide information about weather conditions in the geographic area managed by the functionality.  This information will be in two parts, one for current data and the other for forecast data.  Sensors that are part of other functionality in the Manage Traffic Functional Area will have collected the current data.  The forecast data will have been obtained from an external specialist system.<br/>
