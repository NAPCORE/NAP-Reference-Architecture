It contains data objects that provide details of a route that has been prepared by functionality in the Provide Support for Host Vehicle Systems Functional Area for inclusion in a Freight Transport Execution Plan. It consists of some or all of the following data objects:<br/>- date the implementation of the Freight Transport Execution Plan is to commence,<br/>- departure time from road based Freight Transport Execution Plan origin,<br/>- Freight Transport Execution Plan origin location,<br/>- date the implementation of the Freight Transport Execution Plan to complete (may be same as commence date),<br/>- required arrival time at road based Freight Transport Execution Plan destination,<br/>- FTEP destination location,<br/>- if Freight Transport Execution Plan destination is Freight Terminal Area, facilities required,<br/>- number of Freight Transport Load Items to be moved by the Freight Transport Execution Plan,<br/>- suggested alternative arrival time at the road based Freight Transport Execution Plan destination,<br/>- route details, including turns, distances along each part of the road network, etc.,<br/>- type of Freight Vehicle required.<br/><br/>
