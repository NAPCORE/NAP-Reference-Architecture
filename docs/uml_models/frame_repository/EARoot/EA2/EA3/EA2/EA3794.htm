It contains data objects that provide information about the result of a trip with (optionally) comments from the Traveller that can be used to update of the store of General Trip Preferences (GTP), or the definition of a trip plan implementation for later reuse. Any personal data about the Traveller that this data flow contains must be protected from unauthorised access.<br/>
