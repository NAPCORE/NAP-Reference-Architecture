This data flow only appears in the Context Diagram and contains input from the Multi-Modal System from which data objects are created to provide responses to previous requests for information about transport services provided by operators in other transport modes, e.g. heavy rail, ferries and airlines.  The inputs can also provide data for the management of intersections between the road network and other transport modes, e.g. rail crossings.<br/>
