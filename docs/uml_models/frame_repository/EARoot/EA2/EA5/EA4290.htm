This Data Store shall be used within the Provide Electronic Payments Facilities Functional Area.  It shall contain information objects that provide details of the tariffs of all the services available to the user.<br/><br/>The information object for each tariff shall contain the following data objects:<br/>(1) Service ID (number),<br/>(2) Characteristics of the precise usage done, for example, duration of service usage, location, date, etc. (characters),<br/>(3) Tariff for the characteristic (numbers - in currency),<br/>(4) Type of contract (characters),<br/>(5) Discount rates (numbers - in currency).<br/><br/>The information objects shall be organised so that tariffs for each service is kept together.  For each service there it shall be possible for there to be several characteristics and types of contract each with their own tariff.<br/>
