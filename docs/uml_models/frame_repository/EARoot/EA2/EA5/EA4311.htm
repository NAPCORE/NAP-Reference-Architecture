This Data Store shall be used within the Manage Public Transport Operations Functional Area.  It shall contain information objects that provide the last reported indicators of each PT Vehicle in the fleet.<br/><br/>The information objects in the Store for the static data about each PT Vehicle shall contain the following data objects:<br/>(1) Type (characters),<br/>(2) Vehicle ID (number),<br/>(3) Passenger capacity (number),<br/><br/>The information objects in the Store for the real-time data about each PT Vehicle shall contain the following data objects:<br/>(4) Date (date string),<br/>(5) Time (numbers defining hours, minutes and seconds),<br/>(6) Location (characters and/or numbers, e.g. GPS/Galileo data),<br/>(7) Service ID (number),<br/>(8) Real-time status indicators (numbers and/or characters),<br/>(9) Alarm present (number or character indicating YES or NO),<br/>(10) Number of passengers currently on-board (number),<br/>(11) Passenger alarm raised (number or character indicating YES or NO),<br/>(12) Passenger alarm acknowledged (number or character indicating YES or NO),<br/>(13) Message sent to driver (number or character indicating YES or NO),<br/>(14) Message text sent to driver (text string),<br/>(15) Message received from driver (number or character indicating YES or NO),<br/>(16) Message text received from driver (text string),<br/>(17) Internal video from PT Vehicle (video file).<br/><br/>The data objects that are not provided by the PT Vehicle shall be provided by the PT Operator.  The real-time data shall be provided by the functionality that collects and manages it.  It shall be updated every time a change occurs.<br/>
