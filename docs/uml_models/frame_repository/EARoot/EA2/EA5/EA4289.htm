This Data Store shall be used within the Provide Electronic Payments Facilities Functional Area.  It shall contain information objects that provide a history all the electronic financial transactions performed by the users (Travellers) regarding their use of ITS services.<br/><br/>The information objects for each transaction shall contain the following data objects:<br/>(1) Date of service use (date string),<br/>(2) Place of service use (characters, e.g. nation, region, city, state, area names(s)),<br/>(3) Service ID (number),<br/>(4) Supplementary parameters defining the service (numbers and/or characters),<br/>(5) Contract ID (number),<br/>(6) User ID (number),<br/>(7) ID of organisation with which user places contract (number),<br/>(9) Date of payment (date string),<br/>(10) Mode of payment (characters),<br/>(11) ID of user's account used (number),<br/>(12) ID of operator's account used (number),<br/>(13) Incident notification (characters).<br/>
