This Data Store shall be used within the Provide Electronic Payments Facilities Functional Area.  It shall contain information objects that provide details of the different contracts set up between the users of ITS services and the corresponding operators.  <br/><br/>The information object for each contract shall contain the following data objects:<br/>(1) User ID (number),<br/>(2) ID of organisation with which user places contract (number),<br/>(3) ID's of services covered by the contract (numbers),<br/>(4) Area covered (characters, e.g. nation, region, city, state, area name(s)),<br/>(5) Supplementary parameters defining the service (numbers and/or characters),<br/>(6) Period of validity (date strings for start and end),<br/>(7) Tariffs (optional) (numbers - in currency),<br/>(8) Reduction fees (optional) (numbers - in currency),<br/>(9) Modes of payment (characters),<br/>(10) EP account ID (optional) (numbers).<br/><br/>The Data Store shall enable functions to extract information objects about contracts by specifying the user ID, operator ID, service ID.<br/>
