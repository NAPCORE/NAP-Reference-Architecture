Appears In: DFD F4.7 Provide On-Demand Services
This Data Store shall be used within the Manage Public Transport Operations Functional Area.  It shall contain information objects that are used to provide On-Demand Services and well as details of those Services that have actually been created and delivered.<br/><br/>Each information object in the Store shall contain the following data objects:<br/>(1) Road network data (data for a digital roadmap using a standard format, e.g. GDF)<br/>(2) ID(s) of current PT services used (numbers)<br/>(3) ID(s) of current PT fare schemes used (numbers)<br/>(4) Information about services provided by other transport modes (characters)<br/>(5) Latest and predicted traffic conditions (characters)<br/>(6) Service details, comprising:<br/>    (6a) Service ID (number)<br/>    (6b) Origin (characters and/or numbers, e.g. GPS/Galileo data)<br/>    (6c) Destination (characters and/or numbers, e.g. GPS/Galileo data)<br/>    (6d) Departure date (date string)<br/>    (6e) Departure time (numbers defining hours and minutes)<br/>    (6f) Planned arrival date (date string)<br/>    (6g) Planned arrival time (numbers defining hours and minutes)<br/>    (6h) Stop ID's for pick-up (numbers)<br/>    (6i) Stop ID's for set down (numbers)<br/>    (6j) Planned time at stops (numbers defining hours and minutes)<br/>    (6k) Type of vehicle required (characters)<br/><br/>It shall be possible for each service to be deleted once it has been delivered.<br/>
