Appears In: DFD F9.3 Manage Vehicle access to Sensitive Areas
This Data Store shall be used within the Provide Support for Cooperative Systems Functional Area.  It shall contain information objects that provide records of the Vehicles that have entered a "sensitive area" in the road network. <br/><br/>The information objects in the Store shall contain the following data objects:<br/>(1) Date (date string),<br/>(2) Time (numbers defining hours, minutes and seconds),<br/>(3) Area location (characters, e.g. nation, region, city, state, area names),<br/>(4) Access criteria (characters),<br/>(5) Vehicle ID (number),<br/>(6) Vehicle type (characters),<br/>(7) Hazardous goods on board (characters),<br/>(8) Entry point (characters and/or numbers, e.g. GPS/Galileo data),<br/>(9) Exit point (characters and/or numbers, e.g. GPS/Galileo data).<br/>
