This Data Store shall be used within the Provide Security and Emergency Facilities Functional Area.  It shall contain information objects that provide details of all the information needed to process the data that is produced when any emergency takes place.<br/><br/>Each information object in the Store shall contain the following data objects:<br/>(1) Road network data (data for a digital roadmap using a standard format, e.g. GDF)<br/>(2) Predefined emergency route (data for a digital roadmap using a standard format, e.g. GDF)<br/>(3) Reference number of emergency service to use this route (number)<br/>(4) Description of emergency service to use this route (characters)<br/>(5) Classes of emergency (characters)<br/>(6) Rules to classify an emergency (characters)<br/>(7) The related Road Systems Reference and description for each on (characters)<br/>(8) Procedure to be run for each class of emergency (characters)<br/><br/>It shall be possible for there to be several emergency routes and more than one emergency service that can use each route.  Each class of emergency shall be able to have several rules that enable it to be allocated to a particular incident or accident.<br/>
