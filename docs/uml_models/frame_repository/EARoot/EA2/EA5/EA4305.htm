This Data Store shall be used within the Manage Traffic Functional Area.  It shall contain information objects that provide the data collected about current and predicted incidents.<br/><br/>Each information object in the Store shall contain the following data objects:<br/>(1) Start date (date string),<br/>(2) Start time (date string),<br/>(3) Forecast duration (number - minutes),<br/>(4) Actual duration (number - minutes),<br/>(5) Incident location (characters and/or numbers, e.g. GPS/Galileo data),<br/>(6) Incident type (characters),<br/>(7) Incident severity (characters),<br/>(8) Type and number of vehicles (characters for type(s) and numbers),<br/>(9) Incident strategy used (characters),<br/>(10) Emergency services vehicles used (characters for type(s) and numbers).<br/><br/>The data in some of these entries shall be provided as the incident state changes, whilst in others it shall be updated as the incident impact progresses and remedial action is taken.<br/>
