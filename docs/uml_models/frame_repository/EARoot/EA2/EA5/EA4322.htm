Appears In: DFD F5.13 Provide Vehicle Data and ISA
This Data Store shall be used within the Provide Support for Host Vehicle Services Functional Area.  It shall contain information objects that provide details about the road network.<br/><br/>The information objects in the Store shall contain the following data object:<br/>(1) Area covered (characters, e.g. nation, region, city, state, area names(s)),<br/>(2) Road network data (data for a digital roadmap using a standard format, e.g. GDF),<br/>(3) Link ID (number),<br/>(4) Link start location (characters and/or numbers, e.g. GPS/Galileo data),<br/>(5) Link end location (characters and/or numbers, e.g. GPS/Galileo data),<br/>(6) Number of lanes (number),<br/>(7) Speed limit (number in kph or mph),<br/>(8) Road information from Other Vehicle (characters and numbers).<br/><br/>It shall be possible for these data objects to be updated in real-time as the Vehicle moves through the road network, so that only the data relevant to the area around the Vehicle's current location is stored.<br/>
