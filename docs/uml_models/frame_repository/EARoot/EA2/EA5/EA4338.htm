Appears In: DFD F8.7 Monitor Freight Vehicle during FTEP Implementation
This Data Store shall contain information objects that provide records of the way that each Freight Transport Execution Plan (FTEP) has been implemented.  There shall be one information object for each implementation of each FTEP, thus there will be a record of each implementation of each FTEP.  Each information object shall consist of the following data objects:<br/>(1) FTEP identity (number),<br/>(2) date FTEP implemented (number).<br/>(3) time FTEP implemented (number),<br/>(4) number of Freight Transport Load Item(s) (FTLI(s)) moved (number, must be 1 or more),<br/>(5) date of FTLI pick-up (number),<br/>(6) time of FTLI pick-up (number),<br/>(7) was pick-up on time (yes/no)?<br/>(8) if no, what was the reason for the delay (characters)?<br/>(9) date of FTLI delivery to requested destination (number),<br/>(10) time of FTLI delivery to requested destination (number),<br/>(11) was delivery on time (yes/no)?<br/>(12) if no, what was the reason for the delay (characters)?<br/>(13) where other modes of transport used (yes/no)?<br/>(14) if yes, were these the expected modes (yes/no)?<br/>(15) if no, what was the reason for the use of different other mode(s) of transport (characters)?<br/>(16) did the FTEP have to be modified (yes/no)?<br/>(17) if no, what was the reason for the modification (characters)?<br/>(18) were the traffic conditions on the road network as expected (yes/no)?<br/>(19) if no, what was different about the traffic conditions (characters)?<br/>(20) post implementation comments from the Freight Vehicle Driver (characters).<br/>
