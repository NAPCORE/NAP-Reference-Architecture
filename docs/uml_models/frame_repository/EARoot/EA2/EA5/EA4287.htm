This Data Store shall be used within the Provide Electronic Payments Facilities Functional Area.  It shall contain information objects that provide details of all the accounts used by the users (Travellers) for the electronic payment of services.<br/><br/>The information objects for each account shall contain the following data objects:<br/>(1) User ID (number),<br/>(2) EP account ID's for each user ID (numbers - could be more than one),<br/>(3) Supply for each account (characters),<br/>(4) Current balance (numbers - currency),<br/>(5) Brief history for each account: (characters, but including dates and transaction amounts),<br/>(6) Periods of overdraft (date strings),<br/>(7) Maximum overdraft reached (number - currency).<br/><br/>This information objects hall be loaded, updated and retrieved by several Functions.<br/>
