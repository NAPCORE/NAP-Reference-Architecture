Appears In: DFD F1 Provide Electronic Payment Facilities, DFD F1.5 Access and Credit Control, DFD F1.6 Manage Tariffs and Access Rights
This Data Store shall be used within the Provide Electronic Payments Facilities Functional Area.  It shall contain information objects that provide the access rights for all kinds of users (Including Travellers), all kinds of services, and all environment conditions.  These rights shall be for each user and shall be independent of the financial conditions of the user's account.<br/><br/>The information objects for the access rights to each service shall be structured in the following way:<br/>(1) Service ID (number),<br/>(2) User category (characters),<br/>(3) Environmental condition (characters),<br/>(4) Access rights, which shall be defined as unlimited; limited to some duration, limited to some period, etc. (characters, but may include date strings).<br/><br/>Note that for each service it shall be possible for there to be several user categories and environmental conditions, each with a different set of access rights.  It shall be possible for each set of access rights to consist of one or more entries.<br/>
