Appears In: DFD F6.5.3 Trip Planning
This Data Store shall be used within the Provide Traveller Journey Assistance Functional Area (F6). It shall contain information objects that provide details about the services provided by the PT operations plus the fares that will be charged, and shall be for use in planning the Traveller's trips.<br/><br/>The information objects in the Store for PT services shall contain the following data objects:<br/>(1) ID (number),<br/>(2) Name (characters),<br/>(3) Route (data for a digital roadmap using a standard format, e.g. GDF plus characters and/or numbers, e.g. GPS/Galileo data,<br/>(4) Type of PT Vehicle preferred (characters),<br/>(5) Type of PT Driver preferred (characters),<br/>(6) Type of day on which service operates (characters),<br/>    (6a) Timetable (time in hours and minutes),<br/>    (6b) Frequency (numbers defining hours and minutes),<br/>(7) Variations to the service (characters).<br/><br/>The information objects in the Store for PT service fares shall contain the following data objects:<br/>(8) Fare scheme ID (number),<br/>(9) Fare scheme name (characters),<br/>(10) Date when fare scheme expires (date string),<br/>(11) Time(s) of day when fare scheme operates (numbers defining hours and minutes),<br/>(12) Route ID(s) to which fare scheme applies (numbers),<br/>(13) Type(s) of PT Passenger to which fare scheme applies (characters).<br/><br/>For each type of PT Passenger, the following data objects containing fare information shall be provided:<br/>(14) Range ID's of stop to which fare applies (numbers),<br/>(15) Numbers ID's of stops to which fare applies (numbers).<br/><br/>The above data objects shall be provided by functionality in the Manage Public Transport Operations Functional Area and shall be updated every time a change is made.<br/>
