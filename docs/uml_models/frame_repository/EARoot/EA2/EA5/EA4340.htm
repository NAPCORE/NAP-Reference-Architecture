Appears In: DFD F9.2 Provide Flexible Lane Allocation
This Data Store shall be used within the Provide Support for Cooperative Systems Functional Area.  It shall contain information objects about the links within the road network and use of any Bus Lanes that they contain by non-PT Vehicles. <br/><br/>The information objects in the Store shall contain the following data objects:<br/>(1) Link ID (number),<br/>(2) Link start location (characters and/or numbers, e.g. GPS/Galileo data),<br/>(3) Link end location (characters and/or numbers, e.g. GPS/Galileo data),<br/>(4) Bus Lane present for link (number or character indicating YES or NO),<br/>(5) Bus Lane available for use by other Vehicles (number or character indicating YES or NO),<br/>(6) Historic traffic flow for road link (number in vehicles per hour),<br/>(7) Current traffic flow for road link (number in vehicles per hour),<br/>(8) PT Vehicle use statistics for Bus Lane (numbers),<br/>(9) Other Vehicle use of Bus Lane statistics:<br/>    (9a) Date (date string),<br/>    (9b) Time (numbers defining hours, minutes and seconds),<br/>    (9c) Licence ID (number),<br/>    (9d) Bus Lane ID(s) for which use authorised (numbers),<br/>    (9e) Duration of licence (numbers defining hours, minutes and seconds),<br/>    (9f) Licence suspended (number or character indicating YES or NO),<br/>    (9g) Licence suspension reason(s) (characters),<br/>(10) ID of Bus Lane use requests from other Vehicles refused (number),<br/>(11) Unauthorised Vehicles in Bus Lanes:<br/>    (11a) Date (date string),<br/>    (11b) Time (numbers defining hours, minutes and seconds),<br/>    (11c) Vehicle ID (number),<br/>    (11d) Vehicle type (characters),<br/>    (11e) Vehicle details (image),<br/>    (11f) Bus Lane ID (number).<br/>
