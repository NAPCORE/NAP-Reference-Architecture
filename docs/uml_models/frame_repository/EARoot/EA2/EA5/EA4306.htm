This Data Store shall be used within the Manage Traffic Functional Area.  It shall be split into two parts, one to contain information objects that include the data collected on the use of transport modes and the other to contain information objects that provide demand management strategies.<br/><br/>Each information object in the Store shall contain the following data objects:<br/>(1) Date (date string),<br/>(2) Time (date string),<br/>(3) Transport mode (characters),<br/>(4) Amount of use (numbers as a percentage).<br/><br/>The number of information objects shall be fixed by the number of transport modes, the period over which they are stored<br/><br/>The information objects about the demand re-balancing strategies in the Store shall include the following data objects:<br/>(5) Identity (characters),<br/>(6) Actions (characters).<br/><br/>The number of information objects shall be fixed by the number of strategies.<br/>
