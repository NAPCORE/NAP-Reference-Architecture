Appears In: DFD F8.4 Manage FTEP Preparation
This Data Store shall contain information objects that are produced from inputs provided by the Freight Transport User to the Freight Transport Operations Manager, who may also provide inputs.  The data objects for each Freight Transport Execution Plan shall consist of, but not be limited to, the following:<br/>(1) Freight Transport User identity (number),<br/>(2) number of Freight Transport Load Items (FTLI's) to be moved by this FTEP (number),<br/>(3) for each FTLI:<br/>       (a) location of origin (number),<br/>       (b) date to be collected (number),<br/>       (c) time to be collected (number),<br/>       (d) description of FTLI, e.g. liquid in its own container, liquid requiring a container, powder and hazardous designation (characters),<br/>       (e) size of FTLI, l, w, h (numbers),<br/>       (f) is FTLI to be considered an abnormal load (y/n),<br/>       (g) location of destination (number),<br/>       (h) required date to be delivered to destination (number),<br/>       (i) required time to be delivered to destination (number),<br/>       (j) use of other modes of transport permitted (yes/no),<br/>       (k) any other special requirements (characters).<br/><br/>
