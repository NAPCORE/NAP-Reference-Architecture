This Data Store shall contain information objects that provide details of the capabilities of the various Freight Vehicles that are available for use by the Freight Transport Operations Manager in the implementation of Freight Transport Execution Plans.  There shall be one information object for each Freight Vehicle and this information object shall comprise but not be limited to the following data objects:<br/>(1) vehicle identity - VIN (number),<br/>(2) type of vehicle, e.g. truck, tractor, trailer (characters),<br/>(3) carrying capacity - kg (number),<br/>(4) gross vehicle weight - kg (number),<br/>(5) loaded vehicle dimensions - length, height, width - cm (numbers),<br/>(6) on-board lifting unit available (yes/no)?<br/>(7) if yes, maximum load - kg (number).<br/>(8) type of load carried - packages, container, liquids, powders (characters),<br/>(9) if container, full size or half size (characters)?<br/>(10) currently available (yes/no)?<br/>(11) if no, when expected to be available (number),<br/>(12) type of fuel - petrol/diesel/electric/hybrid (characters).<br/>
