This Function shall be capable of providing the following facilities:<br/><br/>(1) The provision of the HMI for the Emergency Operator so that emergencies and all related information can be managed.<br/>(2) Enable the Emergency Operator to manage the processing, classification and response to incidents or emergencies through Data Flow interfaces to other functionality.<br/>(3) Enable the Emergency Operator to request statistics about the occurrence of incidents and the responses to them.<br/>
