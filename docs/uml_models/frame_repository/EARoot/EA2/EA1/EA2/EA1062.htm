This Function shall be capable of providing the following facilities:<br/><br/>(1)The provision of the HMI for the Communications Operator so that the content of the store of  Public Safety Answering Point (PSAP) Operational Data can be managed.<br/>(2) Enable the Communications Operator to be able to both load new details of a PSAP, or update the existing details of a PSAP in the store of PSAP Operational Data.<br/>(3) Enable the Communications Operator to request and have output to them the details contained in the store of PSAP Operational Data for one or more PSAP's.<br/>

Details:
(a) continuously monitor for the receipt of data flows from the Communications Operator;<br/>(b) when the data flow received in (a) contains data objects providing details about a PSAP, send it to the Manage Store of PSAP Data function to be loaded into the store;<br/>(c) when the data flow received in (a) contains a request for details of one or more PSAP's send the request to the Manage Store of PSAP Data function;<br/>(d) as a result of (c), continuously monitor for receipt of the data flow from the Manage Store of PSAP Data function;<br/>(e) when the data flow in (d) is received, output its contents to the Communications Operator.<br/>
