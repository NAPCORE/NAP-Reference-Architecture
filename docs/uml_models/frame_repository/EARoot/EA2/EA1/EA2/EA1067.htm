This Function shall be capable of providing the following facilities:<br/><br/>(1) Enable a Pedestrian to send an emergency call from a location that is at the roadside.<br/>(2) Provide the Pedestrian with the acknowledgement for the emergency call sent in (1).<br/>(3) Include suitable HMI's the Pedestrian at the roadside.<br/>

Details:
(a) continuously monitor for the receipt of a data flow containing an emergency call from the Pedestrian;<br/>(b) when the data flow in (a) is received, determine the location from the emergency call was made;<br/>(c) send the emergency call to other functionality for processing;<br/>(d) continuously monitor for receipt of a data flow containing an acknowledgement of the emergency message;<br/>(e) when the data flow in (d) is received, output an acknowledgement message to the Pedestrian.<br/>
