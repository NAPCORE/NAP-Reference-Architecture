This Function shall be capable of providing the following facilities:<br/><br/>(1) A HMI for the Travel Information Operator.<br/>(2) The HMI shall enable the Operator to collect data from the store of Private Trip Plan Data about the content of the trip plans that are being produced.<br/>(3) The HMI shall have the ability to format and present the collected data to the Operator as a management report, with the identities and other personal details about the Travellers removed.<br/>

(a) continuously monitor for receipt of the trip plan management report request data flow from the Travel Information Operator;<br/>(b) when the data flow in (a) is received, put the request into the trip plan data request data flow and send it to the Manage Store of Trip Plan Data function;<br/>(c) as a result of (b) continuously monitor for receipt of the trip plan data output data flow;<br/>(d) when the data flow in (c) is received, produce a report about the requested trip plans, making sure that the identities of the Travellers who created and use the trip plans are not included;<br/>(e) output the result of (d) to the Travel Information Operator in the trip plan management report data flow.<br/>
