This High-level Function shall provide facilities for the management of the store pf Private Trip Plan Data.  It shall be able to accept data objects provided by the functionality to enable the trip plan created by the Traveller to be stored.  When the Traveller requests that a previously prepared trip plan is implemented, this Function shall ensure that it is correctly provided to the trip plan implementation and monitoring functionality.<br/>
