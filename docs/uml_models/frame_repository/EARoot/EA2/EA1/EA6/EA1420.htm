Appears In: DFD F6.7 Manage General Trip Preferences, Functional Tree of Area F6 Provide Traveller Journey Assistance
This Function shall be capable of providing the following facilities:<br/><br/>(1) The ability to evaluate the success of the trip plan implementation.<br/>(2) The evaluation shall be based on data provided by the Support Trip functionality and optional input from the Traveller collected after the trip has been completed.<br/>(3) The ability to collect any required input as comments on the trip and on the support given using the Define Traveller's General Trip Preferences functionality. <br/>(4) The ability to send the results of the analysis of the trip performance for output to the Travel Information Operator through the Enable Operator Access to General Trip Preferences functionality.<br/>

Details:
(a) when the trip completion report for evaluation data flow is received, send the request post trip preferences data flow to the Define Traveller's General Trip Preferences function;<br/>(b) if and when the post trip preferences data flow is received, analyse its contents for inclusion in the trip evaluation;<br/>(c) the evaluation of the trip performance shall be carried out using the input from (a) and (b);<br/>(d) if necessary the evaluation in (c) shall be carried out without any input from the Traveller in (b);<br/>(e) the results of (d) shall show how well the trip was performed, looking at parameters such as the journey time performance, relevance and timeliness of information provided to the Traveller during the trip, and the success with which connections between different modes of transport were made;<br/>(f) the output from (e) shall be sent to the Enable Operator Access to General Trip Preferences in the trip evaluation report for output data flow.<br/>
