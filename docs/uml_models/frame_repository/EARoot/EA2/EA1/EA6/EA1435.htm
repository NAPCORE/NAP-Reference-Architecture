This Function shall be capable of providing the following facilities:<br/><br/>(1) Manage the contents of the Road Trip Planning Data Store.<br/>(2) Enable data objects containing road trip planning such as road geometry that are received to be loaded into the Data Store in a way that does not interfere with the data reading process and fuses it with data objects already in the Store.<br/>(3) Enable data objects containing road trip planning to be retrieved from the Data Store on request in an orderly manner that does not interfere with the data loading process.<br/>
