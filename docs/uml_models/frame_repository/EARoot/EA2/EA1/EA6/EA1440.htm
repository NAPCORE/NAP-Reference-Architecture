This High-level Function shall be responsible for the management of the store of General Trip Preferences (GTP) that the Traveller specifies for use in every trip plan that they create.  It shall enable the Traveller to set up these preferences and for the Travel Information Operator to be able to view them in an anonymous way, i.e. without knowing the identity of any of the Travellers to which they belong.  The Function shall also enable the Traveller to modify their preferences as a result of completing the implementation of a trip plan and for a report about the performance of the trip plan to be automatically provided to the Travel Information Operator.<br/>
