This Function shall be capable of providing the following facilities:<br/><br/>(1) A HMI through which the PT Operator may have access to the criteria being used to create travel plans for Vehicle Sharers.<br/>(2) The HMI shall enable the PT Operator to request and be provided with output showing the current criteria, and for the Operator to amend the criteria,<br/>(3) The HMI shall not enable the PT Operator to see any information about Vehicle Sharers.<br/>
