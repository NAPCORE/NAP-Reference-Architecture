This Function shall be capable of providing the following facilities:<br/><br/>(1) The ability to monitor for the receipt of alarms raised by PT Passengers on- board the PT Vehicle.<br/>(2) The ability to forward details of the alarms to the Monitor PT Vehicle functionality,<br/>(3) A suitable HMI through which the responses to alarms can be displayed to PT Passengers on- board the PT Vehicle.<br/>
