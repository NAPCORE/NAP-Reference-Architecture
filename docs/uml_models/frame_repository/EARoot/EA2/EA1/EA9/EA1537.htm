This Function shall be capable of providing the following facilities:<br/><br/>(1) The ability to manage the non-PT Vehicles that wish to obtain licences to use Bus Lanes.<br/>(2) The ability to receive the request for using Bus Lanes from the Driver interface functionality and collect data about the Vehicle characteristics, such as its speed and current location, from its own interface and sensors.<br/>(3) When all the data has been accumulated, the ability to send the Bus Lane use request as a licence request to the functionality responsible for managing the use of Bus Lanes and to pass on the response to the Driver interface functionality.<br/>(4) If the response contains a licence, then the ability to monitor the location of the Vehicle as it moves through the road network and provide messages for output to the Driver through the Driver interface functionality when the Vehicle correctly using any Bus Lanes.<br/>

Details:
(a) when the bus lane request data flow is received, check its contents for missing elements if not already available, collect the data from Vehicle Systems, Location Data Source and Map Data;<br/>(b) process the location data to determine the location of the Vehicle to within a suitable tolerance that enables it to be located with a particular lane of a particular segment of the road network;<br/>(c) identify the lane and road segment in (b) using the map data;<br/>(d) if the destination is not available in the provide destination for bus lane use data flow, send a request for it in the other bus lane information data flow and continuously monitor for receipt of it in the request bus lane use data flow;<br/>(e) include the data from (a), (b), (c) and (d) in the bus lane request data flow;<br/>(f) when the bus lane licence response data flow is received, pass on its contents to the Driver Interface function, noting the result for future reference;<br/>(g) if the bus lane licence revoked data flow is received, check that a licence acceptance data flow has been previously received and that the licence is still valid;<br/>(h) if the result of (g) is that the revocation applies to a current licence then pass it on through the data flow to the Driver Interface function;<br/>(i) if the warning data flow is received, repeat the check in (g);<br/>(j) if the result of (i) is that a licence has not been obtained to use the bus lave in which the vehicle is travelling then send the warning message to the Driver Interface function.<br/>
