This Function shall be capable of providing the following facilities:<br/><br/>(1) A HMI through which the Road Network Operator can define and manage the use of Bus Lanes by other (non-PT) Vehicles.<br/>(2) The HMI shall enable the Operator to provide and/or amend the data about the road network and in particular the allocation of Bus Lanes, including the ability to temporarily make some Bus Lane segments unavailable for by other non-PT Vehicles.<br/>(3) The HMI shall also enable the Operator to obtain statistics about the use of the Bus Lanes.<br/>
