This Function shall be capable of providing the following facilities:<br/><br/>(1) A HMI through which the Road Network Operator can have access to the criteria used to determine routes for Vehicles that are carrying Hazardous Goods.<br/>(2) The ability for the Operator to use the HMI to either be provided with an output of the current criteria, or to update the current criteria.<br/>
