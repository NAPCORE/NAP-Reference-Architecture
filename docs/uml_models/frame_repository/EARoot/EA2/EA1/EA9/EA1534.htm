This Function shall be capable of providing the following facilities:<br/><br/>(1) A HMI through which the Drivers can request priority for their Vehicle through the urban road network and receive the responses to these requests.<br/>(2) The HMI must require the Driver to indicate the destination and any constraints, e.g. mandatory "way points", so that a green wave route can be created.<br/>(3) The ability to send the data provided by the Driver through the HMI to the Priority Request functionality. <br/>(4) The HMI must output the data from the Priority Request functionality as soon as it is received, superseding for the duration of its relevance any other outputs that the Driver may be receiving from other functionality.<br/>
