This Function shall be capable of providing the following facilities:<br/><br/>(1) The ability to obtain the identity of the violator, i.e. the person who has committed the violation, from the store of User's Registration Data using the identification of the violating Vehicle as the reference for the data.<br/>(2) The ability to send this extracted information to other functionality in the Provide Support for Law Enforcement Functional Area.<br/>

(a) when the trigger input data flow is received extract the identity of the violating Vehicle;<br/>(b) do not change the type of violation in the other input data flow that will also be received in (a);<br/>(c) search in the store of User's Registration Data for the identity of the owner or recorded driver of the Vehicle using the second other output data flow;<br/>(d) receive the data resulting from (c) in the second other input data flow;<br/>(e) send the result of (d) to the Sort Violations Notifications Function, using the first trigger output data flow;<br/>(f) include with the data flow in (e) the first other output data flow containing the type of violation, which must be a copy of the first other input data flow received in (b).<br/>
