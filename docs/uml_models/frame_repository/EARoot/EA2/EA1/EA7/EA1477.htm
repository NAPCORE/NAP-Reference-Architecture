This Function shall be capable of providing the following facilities:<br/><br/>(1) The ability to update the store of User's Registration Data (D7.2) whenever an update is received, or after making a request for the Law Enforcement Agency to provide it.<br/>(2) The ability to make sure that the registrations concern the identification of users, the ownership or utilisation relationship between users and vehicles, and the registration of operators allowed to perform commercial operations using the road network.<br/>(3) The ability to respond to requests for data from the store of Rules Data.<br/>

Details:
(a) continuously monitor for receipt of data flows from the Law Enforcement Agency;<br/>(b) if none of the data flows in (a) are received, request that any new or updated user registrations are provided from the relevant Law Enforcement Agency;<br/>(c) when any data flows in (a) are received, load the new registration data or updates of existing registration data into the store of User's Registration Data;<br/>(d) continuously monitor for receipt of data flows requesting details of users registrations;<br/>(e) when a data flow in (d) is received, extract the required data objects from the store of User's Registration Data and send them to the functionality from which the data flow was received.<br/>
