This Function shall be capable of providing the following facilities:<br/><br/>(1) A HMI through which driving instructions and Estimated Time of Arrival (ETA) are output to the Driver.<br/>(2) The HMI shall be able to output both sets of information continuously, updating the output immediately fresh data is received.<br/>(3) If no data is received from which to generate the output, the HMI shall be able to output a warning message to the Driver to indicate that there is nothing to display.<br/>
