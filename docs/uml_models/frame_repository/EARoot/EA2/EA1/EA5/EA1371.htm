This High-level Function shall enable a trip plan to be checked to see if total time exceeds the maximum time a Driver can work without taking a break.  If no break is needed, the trip plan will be sent to the Function that manages the use of Un/Loading Zones.  When the length of the trip plan requires a break to be taken, a suitable rest area will be found along the route specified in the trip plan.<br/>
