This High-level Function shall enable data objects to be created from the Host Vehicle's systems interfaces, data objects created by its functionality to be made available to the Host Vehicle's systems and to enable communications between the Host Vehicle and Other Vehicles, in order to support the provision of advanced Driver assistance systems by systems in the Host Vehicle.  It shall provide functionality to enable copies of the current traffic signals to be displayed to the Host Vehicle Driver, the Host Vehicle identity (ID) to other functions, and shall inform the relevant authorities when the Host Vehicle is being used illegally.<br/>
