This Function shall be capable of providing the following facilities:<br/><br/>(1) The ability to take responsibility for processing information from sensors that detect the atmospheric conditions near to the Host Vehicle, i.e. the Vehicle in which it is operating.<br/>(2) The ability to process the sensor data to determine such things as temperature, wind strength and direction (taking account of the Host Vehicle's direction of travel), the presence of rain, etc.<br/>(3) Once processing is complete, the ability to send the data in digital form to the Determine and store local Host Vehicle Conditions Function.<br/>
