Appears In: DFD F5.15 Provide In-vehicle Detection, Functional Tree of Area F5 - High-level Function F5.15 only
This Function shall be capable of providing the following facilities:<br/><br/>(1) The ability to take responsibility for the output of particular warning messages to the Driver of the Host Vehicle, i.e. the Vehicle in which it is operating.<br/>(2) A HMI through which these messages can be output to the Driver of the Host Vehicle.<br/>(3) The ability for these messages to warn the Driver that the Host Vehicle is being driven in an unsafe manner, or that it is about to collide with another Vehicle that is nearby.<br/>(4) The ability for the outputs of both messages to be made simultaneously and for possible corrective action to be suggested to the Driver.<br/>
