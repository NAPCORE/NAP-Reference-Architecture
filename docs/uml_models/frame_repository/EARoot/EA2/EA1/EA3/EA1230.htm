This Function shall be capable of providing the following facilities:<br/><br/>(1) A HMI through which the Parking Operator can manage one or a number of car parks.<br/>(2) The HMI shall enable the Parking Operator to set up data about car parks that is loaded into the store of Car Park Data, to provide output of the current static data and to provide output of the current car park data (occupancy and/or number of spaces).<br/>(3) The HMI shall also enable the Parking Operator to override the current status so that for example, a car park can be set to "CLOSED" regardless of how many of its spaces are not being used.<br/>
