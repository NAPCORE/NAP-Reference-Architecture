This Function shall be capable of providing the following facilities:<br/><br/>(1) The ability to provide up-to-date information for digital maps and databases for segments of the inter-urban road network.<br/>(2) The ability for the information provided by this Function to include structural alteration, static speed limits and default journey times.<br/>(3) The ability to provide updated information to the digital map provider in order to be implemented in the next issue of digital maps as well as to in-vehicle devices for use in current and future planned journeys.<br/>
