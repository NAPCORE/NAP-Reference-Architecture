This Function shall be capable of providing the following facilities:<br/><br/>(1) An HMI through which the Tunnel Operator can monitor and manage the operation of tunnels in the road network managed by the System.<br/>(2) The ability of the HMI to inform the Tunnel Operator of any exceptional situations that have been detected, e.g. congestion, blocked lane, fire and pollution and of the action being automatically taken in response.<br/>(3) If the Tunnel Operator is advised that there are no pre-defined actions that can be taken, the HMI shall enable them to be defined by the Operator for immediate implementation.<br/>(4) The ability for the Tunnel Operator to set up actions in advance of the detection of any exceptional situations.<br/>

(a) continuously monitor for receipt of data flows from the Tunnel Operator;<br/>(b) when the data flow in (a) is received, check its contents and if they list the actions to be automatically taken, output them to the Tunnel Operator in the exceptional condition action taken data flow;<br/>(c) as a result of (b) continuously monitor for receipt of the exceptional condition action response data flow and when received send its contents to the Assess Tunnel Status and Take Action function in the tunnel action responses data flow;<br/>(d) if the data flow in (a) contains a request for new actions to be provided, send this to the Tunnel Operator in the exceptional condition action request data flow;<br/>(e) as a result of (d), monitor for receipt of the exceptional condition action request response data flow from the Tunnel Operator;<br/>(f) when the data flow in (e) is received, send the new actions to the Assess Tunnel Status and Take Action function in the tunnel action definitions data flow;<br/>(g) continuously monitor for receipt of the tunnel action definitions data flow from the Tunnel Operator;<br/>(h) when the data flow in (g) is received, send its contents to the Assess Tunnel Status and Take Action function in the tunnel action definitions data flow.<br/>
