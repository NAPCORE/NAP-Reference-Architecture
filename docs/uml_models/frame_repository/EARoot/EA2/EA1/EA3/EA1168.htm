This High-level Function shall be responsible for the actual output of messages and that status of signs that can be seen by Drivers using all or part(s) of the inter-urban road network.  It shall be able to use these messages and signs to manage the use if the inter-urban road network by Vehicles.  This shall include restricting the use of lane, setting speed limits, assigning particular types of Vehicles to particular lanes and other measures that will help to optimise and improve the efficiency with which the inter-urban road network is used, as well as reducing the impact of Vehicles on the environment.  The Function shall enable the current strategies to be temporarily overridden in order that Emergency Vehicles can be given green waves, or part(s) of the inter-urban road network can be "coned" off to enable the Emergency Services to respond to incidents.    If a Vehicle violates one or more of any mandatory messages and/or sign states, it shall be possible for its details to be sent to the functionality that will forward them to the Law Enforcement Agency.  It shall also be possible for the Road Network Operator to have access to the functionality to both monitor what messages and sign states are being output and to override some or all of those outputs if necessary.<br/>
