This Function shall be capable of providing the following facilities:<br/><br/>(1) The ability to check the status of a user s (Traveller or Driver) account.<br/>(2) If an overdraft notification has been received for this account (or for several different accounts with the same owner), the ability to include the account(s) in the "black list", which means that the user shall not be allowed to use this account until sufficient funds have been provided to rectify the situation, unless an  agreement  message has been received.<br/>

Details:
(a) process the status changes of user's account: if an overdraft notification received write the account number in the black list, or conversely, if new account balance is above a certain threshold, remove it from the black list;<br/>(b) if an agreement message is received, indicate that the user is allowed to use the service;<br/>(c) send the fraud notification to the Provide Support for Law Enforcement Functional Area;<br/>(d) register s all incorrect payments (frauds) in the store of Incorrect Payment Data, and initiate a block on the use of that service.<br/>
