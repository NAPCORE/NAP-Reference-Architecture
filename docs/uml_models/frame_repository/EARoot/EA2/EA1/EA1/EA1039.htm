This Function shall be capable of providing the following facilities:<br/><br/>(1) The ability to manage the store of "tariffs" Data.<br/>(2) The ability to load the store with details of PT fares received from functionality in the Manage Public Transport Functional Area and/or details of charges for road use from functionality in the Manage Traffic Functional Area.<br/>(3) The ability to define within the store, the structure in which the "tariffs" data will be stored, with different structures being possible for PT fares and charges for road use.<br/>
