This is Data Flow Diagram 0 (DFD0).  It is the highest level of Data Flow Diagram (DFD) in the FRAME Functional View.  DFD0 contains all of the Functional Areas, the data flows that link them together and the data flows that link the Functional Areas to the Terminators in the Context Diagram.  THe Functional Areas contain all of the functionality needed to provide what is defined in the User Needs.<br/><br/>
