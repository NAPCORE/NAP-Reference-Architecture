This Function shall be capable of providing the following facilities:<br/><br/>(1) Manage the store of previously prepared Freight Transport Execution Plans.<br/>(2) Update the store in (1) as and when new Plans are created and approved by other functionality in the Manage Freight and Fleet Operations Functional Area.<br/>(3) Provide approved Plans for implementation by other functionality in the Manage Freight and Fleet Operations Functional Area.<br/>(4) Respond to requests by providing details of previously prepared Freight Transport Execution Plans.<br/>

Details:
(a) continuously monitor for receipt of the FTEP data request data flow from the Provide FTOM Interface function;<br/>(b) when the data flow in (a) is received, extract the requested FTEP data from the store and return it to the Provide FTOM Interface function in the  data flow requested FTEP data;<br/>(c) continuously monitor for receipt of the existing FTEP data flow from the Provide FTU Interface function;<br/>(d) when the data flow in (c) is received, extract the requested FTEP data from the store and return it to the FTU Interface function in the existing FTEP data data flow;<br/>(e) continuously monitor for the request FTEP details data flow from the Implement FTEP function;<br/>(f) when the data flow in (e) is received extract the FTEP data from the store and return it to the Implement FTEP function in the requested FTEP details data flow;<br/>(g) continuously monitor for receipt of the new FTEP for store data flow and when received, load the content into the store creating a new FTEP entry in the process;<br/>(h) continuously monitor for receipt of the request existing FTEP details data flow from the Prepare FTEP function;<br/>(i) when the data flow in (e) is received extract the FTEP data from the store and return it to the Prepare FTEP function in the requested existing FTEP details data flow;<br/>(j) always ensure that there is no conflict between the data flows that load data for a new FTEP into the Data Store and the data flows that read data for an existing FTEP from the Data Store. <br/>
