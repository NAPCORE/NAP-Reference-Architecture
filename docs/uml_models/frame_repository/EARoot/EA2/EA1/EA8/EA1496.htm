Appears In: Data Flow Diagram for all Functional Areas (DFD0), Functional Tree of Functional Area F8 Manage Freight and Fleet Operations
This Functional Area shall provide functionality that enables goods (called Freight Transport Load Items) to be moved by Freight Vehicles using the road network.  However it shall enable the option for other non-road based modes of transport to be used for one or more parts of the movement.  The way in which the goods are to be moved shall be described in a Freight Transport Execution Plan which shall be prepared by some of the functionality in this Functional Area using data about its origin and destination provided by the Freight Transport User.  It shall be possible for Freight Transport Execution Plans to be prepared in advance and for existing Plans to be re-used and modified to suit the changing requirements of the Freight Transport User.  The implementation of a Plan by some of the functionality in this Functional Area, with progress reports being made available to the Freight Transport User and the Freight Transport Operations Manager, who manages the implementation on behalf of the Freight Transport Service Provider.  If problems develop with the implementation of a Plan then at the direction of the Freight Transport Operations Manager it shall be possible for a revised Plan to be prepared to start from the point reached by the Plan that is being implemented.<br/><br/>Mnemonic: mffo<br/>
