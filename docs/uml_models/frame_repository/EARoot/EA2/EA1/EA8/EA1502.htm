This Function shall be capable of providing the following facilities:<br/><br/>(1) The ability to collect data objects containing details of the road network, such as geometry and location of road segments, junctions, etc.<br/>(2) The ability to receive from other functionality data objects that provide the current location of the Freight Vehicle as given by sensors on the Vehicle.<br/>(3) The ability to receive from other functionality data objects that provide the expected location of the Freight Vehicle according to the Freight Transport Execution Plan.<br/>(4) The ability to compare the locations provided by (2) and (3) above and from this generate a status update, e.g. the Freight Vehicle is/is not where it is expected to be according to the Freight Transport Execution Plan.<br/>(5) The ability to provide other functionality with the location of the Freight Vehicle for output to the Freight Transport Operations Manager and for use in reporting its status. <br/>(6) The ability to predict the time of arrival (PTA) of the Freight Vehicle at multi-modal interchange points and to send data objects containing these predictions to other functionality.<br/>

Details:
(a) continuously monitor for receipt of the map data for FTEP monitoring data flow from the Geographic Information Provider Terminator;<br/>(b) when the data flow in (a) is received, update the functions' internal data store with the received digital map data;<br/>(c) continuously monitor for receipt of the FV expected location FTEP data flow from the Implement FTEP function and the vehicle location for status data flow from the Collect Data from Freight Vehicle function;<br/>(d) when both of the data flows in (c) have been received, compare the locations and decide if the Freight Vehicle as at the location expected from the content of the FTEP;<br/>(e) if it is decided in (d) that the Freight Vehicle is not at the expected location send a predicted arrival time (pta) to the Provide Interface to Other Transport Modes function in the FV pta for other mode;<br/>(f) whatever the outcome of the decision in (d) send the current location of the Freight Vehicle to the following:<br/>      - the Calculate FTEP Implementation Status function in the FV road network location data flow, and<br/>      - the Provide FTOM interface for FTEP Monitoring function in the vehicle location status data flow.<br/>
