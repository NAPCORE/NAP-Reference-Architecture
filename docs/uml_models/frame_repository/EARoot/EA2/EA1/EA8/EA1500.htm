This High-level Function shall collect and manage data objects from sensors and systems on the Freight Vehicle.  It shall use the collected data objects to provide information about the Freight Vehicle as it performs the actions required of it in the Freight Transport Execution Plan.  As the Freight Transport Execution Plan is implemented, the Function shall be able to provide status updates on request to the Freight Transport User and the Freight Transport Operations Manager and shall create a record of the progress of the implementation.  If the implementation does not proceed as expected then the Function shall be able to request a change in the time of the handover to another transport mode (if used) and in extreme instances that a new Freight Transport Execution Plan is prepared.<br/>
