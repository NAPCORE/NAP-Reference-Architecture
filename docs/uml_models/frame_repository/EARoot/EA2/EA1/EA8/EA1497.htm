This High-level Function shall provide interfaces for the Freight Transport User and the Freight Transport Operations Manager.  The Freight Transport User shall be able to request the creation of Freight Transport Execution Plans to fulfil its need to move Freight Transport Load Items.  These Plans can be based on any that have been previously created, or completely new.  The Freight Transport Operations Manager shall be able to manage the creation of new Freight Transport Execution Plans, including the review of those previously created.<br/>
