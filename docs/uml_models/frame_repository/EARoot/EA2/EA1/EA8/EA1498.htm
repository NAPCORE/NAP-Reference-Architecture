This High-level Function shall enable Freight Transport Execution Plans to be prepared.  The preparation process shall take place at the direction of the Freight Transport Operations Manager and use data objects containing details of the Freight Transport Load Items to be moved, i.e. origin, destination, etc. provided by the Freight Transport User.  Data objects about the road network layout, travel conditions in it, services provided by other modes of transport and the weather shall be provided by other functionality and particular Terminators.  Once preparation of a new Freight Transport Execution Plan has been completed, it shall be sent to other functionality for storage so that it can be used at a later point in time when requested by the Freight Transport User.<br/>
