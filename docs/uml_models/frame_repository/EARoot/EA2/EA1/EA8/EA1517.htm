This Function shall be capable of providing the following facilities:<br/><br/>(1) Manage the content of the store of Freight Terminal Area Facilities Data.<br/>(2) Periodically request an update of the content of the Data Store from the Freight Terminal Area Facilities Terminator.<br/>(3) When requested by the Freight Transport Operations Manager provide details of the facilities available at one or more Freight Terminal Areas which the Manager can then include with the data objects used to prepare a Freight Transport Execution Plan.<br/>(4) When requested by other functionality in the Manage Freight and Fleet Operations Area provide details of the facilities available at one or more Freight Terminal Areas for use in the implementation of a Freight Transport Execution Plan.<br/>
