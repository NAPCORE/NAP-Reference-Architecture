This Function shall be capable of providing the following facilities:<br/><br/>(1) The ability to manage the store of Freight Transport Execution Plan Implementation Records so that its content is stored and accessed in an efficient and timely manner, with the records for each Freight Transport Execution Plan being kept separately.<br/>(2) The ability to receive data objects from other functionality containing details of how the implementation of a Freight Transport Execution Plan is progressing and to load these data objects into the store of Freight Transport Execution Plan Implementation Records.<br/>(3) The ability to respond to data objects sent from other functionality that request the records which were created as a result of the implementation of particular Freight Transport Execution Plans.<br/>(4) The ability to deduce the road network performance from the records of the implementation of one or more Freight Transport Execution Plans and to send the results to other functionality. <br/>(5) The ability to deduce the performance of other modes of transport from the records of the implementation of one or more Freight Transport Execution Plans and to send the results to other functionality.<br/>

Details:
(a) continuously monitor for receipt of the following data flows:<br/>- the FTEP implementation status for record data flow from the Calculate FTEP Implementation Status function,<br/>- the FTEP implementation driver record data flow from the Provide Freight Vehicle Driver Interface function,<br/>- the FTEP Implementation record from the Implement FTEP function <br/>(b) when any of the data flows in (a) are received load their contents into the FTEP Implementation Record Data Store for the specified FTEP;<br/>(c) also continuously monitor for receipt of the request FTEP implementation record data flow from the Provide FTOM Interface for FTEP Monitoring function;<br/>(d) when the data flow in (c) is received, retrieve the requested FTEP implementation record(s) from the FTEP Implementation Record Data Store and send them to the Provide FTOM Interface for FTEP Monitoring function in the requested FTEP implementation record data flow;<br/>(e) periodically, or when the implementation of a particular FTEP has been completed, calculate the way that the road network has been used, highlighting such data as the average speed of the Freight Vehicle and when the Vehicle was stationary, and send the results to the Collect Road Network Data function in the deduced road network performance data flow;<br/>(f) again periodically, or when the implementation of a particular FTEP has been completed and only if other mode(s) of transport were used, calculate the way that the other mode(s) were  used, highlighting such data as the time spent being moved by the other mode(s) and send the results to the Collect FTEP Data from FTOM and MMS function in the deduced other mode performance data flow.<br/>
