This Function shall be capable of providing the following facilities:<br/><br/>(1) The ability to receive inputs generated by equipment and/or sensors on-board a Freight Vehicle that will enable the following to be determined:<br/>- the status of equipment used to carry and/or load/unload Freight Transport Load Items onto/off the Freight Vehicle,<br/>- the current speed of the Freight Vehicle,<br/>- the current location of the Freight Vehicle,<br/>- the current status of the Freight Vehicle.<br/>(2) The ability to convert the inputs received in (1) into data objects that show the parameters identified in it and to add to each of those parameters, as a minimum, the following data objects:<br/>- date,<br/>- time,<br/>- Freight Vehicle identity, e.g. VIN.<br/>(3) The ability to send the data objects created in (1) and (2) above to other functionality for storage and analysis.<br/>
