This Function shall be capable of providing the following facilities:<br/><br/>(1) The ability to manage the store of Incident Response Data so that its content is stored and accessed in an efficient and timely manner.<br/>(2) The ability to receive from other functionality data objects containing the required response to particular identifiable types of incident and load them into the store of Incident Response Data.<br/>(3) The ability to respond to data objects containing requests for the response to one or more particular types of incident to be retrieved from the store of Incident Response Data and sent to the functionality that made the request.<br/>(4) If no pre-defined response is found for the requested type of incident the ability to send data objects indicating this to the functionality that provides the interface for the Freight Transport Operations Manager.<br/>

(a) continuously monitor for receipt of the request incident response data flow from the Implement FTEP function;<br/>(b) when the data flow in (a) is received find the incident response that fits with the incident from the Incident Response Data Store and send it in the requested incident response data flow to the Implement FTEP function;<br/>(c) if in (b) no suitable incident response can be found in the Incident Response Data Store send an indication that there is no suitable response in the no incident response data flow to the Provide FTOM Interface for FTEP Monitoring function;<br/>(d) as a result of (c) continuously monitor for receipt of the new incident response data flow from the Provide FTOM Interface for FTEP Monitoring function and when it is received send its contents to the Implement FTEP function in the requested incident response data flow and load the new incident response into the Incident Response Data Store;<br/>(e) continuously monitor for receipt of the request incident response details data flow from the Provide FTOM Interface for FTEP Monitoring function;<br/>(f) when the data flow in (e) is received, find the requested incident response details from the Incident Response Data Store and send them in the incident response details data flow to the Provide FTOM Interface for FTEP Monitoring function.<br/>
