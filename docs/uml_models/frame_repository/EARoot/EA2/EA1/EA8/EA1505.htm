This Function shall be capable of providing the following facilities:<br/><br/>(1) The ability to provide an interface through which audio and/or visual information can be exchanged with the Freight Transport Operations Manager.<br/>(2) The ability to convert data objects received from other functionality into audio and/or visual outputs that can be easily understood by the Freight Transport Operations Manager.<br/>(3) The ability to receive audio and/or visual inputs from the Freight Transport Operations Manager and convert them into digital objects that can be sent to other functionality for their interpretation and use.<br/>(4) The ability to alert the Freight Transport Operations Manager to the output of audio and/or visual messages.<br/>

(a) continuously monitor for receipt of the message from Driver data flow from the Provide Freight Vehicle Driver Interface function;<br/>(b) when the data flow in (a) is received, output its content in the message from Driver data flow to the Freight Transport Operations Manager Terminator in audio and/or visual form;<br/>(c) continuously monitor for receipt of the request incident responses data flow from the Freight Transport Operations Manager Terminator and, when it is received, send the request to the Manage Incident Response Data Store function in the request incident response details data flow;<br/>(d) as a result of (c), continuously monitor for receipt of the incident response details data flow from the Manage Incident Response Data Store function and, when it is received, output its contents to the Freight Transport Operations Manager Terminator in the requested incident responses data flow;<br/>(e) continuously monitor for receipt of the no incident response data flow from the Manage Incident Response Data Store function and, when it is received , output its contents to the Freight Transport Operations Manager Terminator in the incident response needed data flow;<br/>(f) as a result of (e), continuously monitor for receipt of the incident responses data flow from the Freight Transport Operations Manager Terminator and, when it is received, send its contents to the Manage Incident Response Data Store function in the new incident response details data flow;<br/>(g) continuously monitor for receipt of the message for Driver data flow from the Freight Transport Operations Manager Terminator and, when received, send its contents to the Provide Freight Vehicle Driver Interface function in the message for Driver data flow;<br/>(h) continuously monitor for receipt of the other mode service issues data flow from the Implement FTEP function and, when it is received, output its content in the other mode service issue data flow to the Freight Transport Operations Manager Terminator;<br/>(i) as a result of (h), continuously monitor for receipt of the other mode negotiation request data flow from the Freight Transport Operations Manager Terminator and, when it is received, send its content to the Provide Interface for Other Transport Modes function in the request alternative services data flow;<br/>(j) as a result of (i) continuously monitor for receipt of the alternative services response data flow from the Provide Interface for Other Transport Modes function and, when it is received, output its content to the Freight Transport Operations Manager Terminator in the other mode negotiation response data flow;<br/>(k) continuously monitor for receipt of the prepare revised FTEP data flow from the Freight Transport Operations Manager Terminator and, when it is received, send the request to the Prepare Freight Transport Execution Plan for Implementation Function in the prepare revised FTEP data flow;<br/>(l) continuously monitor for receipt of the request FTEP implementation record data flow from the Freight Transport Operations Manager Terminator and, when it is received, send its contents to the Evaluate FTEP Implementation function in the request FTEP implementation record data flow;<br/>(m) as a result of (l) continuously monitor for receipt of the requested FTEP implementation record data flow from the Evaluate FTEP Implementation function and, when it is received, output its contents to the Freight Transport Operations Manager Terminator in the requested FTEP implementation record data flow;<br/>(n) continuously monitor for receipt of the request FV status data flow from the Freight Transport Operations Manager Terminator and, when it is received, send the FV status request for FTOM data flow to the Calculate FTEP Implementation Status function;<br/>(o) as a result of (p)  continuously monitor for receipt of the FV status for FTOM data flow from the Calculate FTEP Implementation Status function and, when it is received, output its content to the Freight Transport Operations Manager Terminator in the requested FV status data flow;<br/>(p) continuously monitor for receipt of the FTEP progress status from the Calculate FTEP Implementation Status function and, when it is received, output its content to the Freight Transport Operations Manager Terminator in the FTEP progress report data flow.<br/>
