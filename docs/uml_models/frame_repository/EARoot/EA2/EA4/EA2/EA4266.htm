This Terminator is part of the Vehicle Terminator and shall represent the systems that are in the Vehicle and which are provided by the Vehicle manufacturer for the basic control and management of Vehicle operations.  It is expected that this Terminator shall provide data objects to the functionality in the system either directly through an interface or through sensors, and these shall be used to manage the operation of the road network.  In addition, the Terminator shall be sent data objects through an interface by functionality in the system that requests action to be taken by the Vehicle Systems.  Note that this shall be a "request" and not a command, as it shall not be possible for the Vehicle to be controlled by the functionality in the system.<br/><br/>Mnemonic: v.vs<br/>
