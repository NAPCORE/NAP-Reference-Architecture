This Terminator is part of the Driver Terminator and shall represent a human entity that drives Vehicles used to provide On-Demand Services for Travellers.  It is expected that the Terminator shall provide inputs from which data objects can be created by system functionality to represent requests and data for use in the system and be capable of translating into human readable form of data objects output from the functionality in the system.  The vehicle type for this Driver is defined in the description of the On-Demand Service Vehicle Terminator.  The functionality in the system must also be able to communicate with more than one human entity that is an On-Demand Service Driver, each of which may belong to the same organisation, or to different organisations that operate On-Demand Services.<br/><br/>Mnemonic: d.odsd<br/>
