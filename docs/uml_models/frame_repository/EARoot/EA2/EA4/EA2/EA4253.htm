This Terminator is part of the External Service Provider Terminator and shall represent an organisation that manages information about Drivers and Vehicles.  It shall provide data objects containing Vehicle identification to assist the functionality that checks raw Floating Card Data (FCD).  The Terminator shall also provide data objects containing Driver and Vehicle identification to assist the functionality that provides incident responses in response to a request from that functionality.  All data objects provided by this Terminator must comply with the requirements of the General Data Protection Regulation (GDPR - EU2016/679 and any subsequent amendments).<br/><br/>Mnemonic: esp.dvip<br/>
