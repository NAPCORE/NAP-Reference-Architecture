This Terminator is part of the Driver Terminator and shall represent a human entity that drives a vehicle belonging to one of the Emergency Services, i.e. an Emergency Services Vehicle.  It is expected that the Terminator shall provide inputs from which data objects can be created by system functionality to represent requests and data for use in the system and be capable of translating into human readable form data objects output from the functionality in the system.  The Vehicle type is defined in the description of the Emergency Vehicle Terminator.  The functionality in the system must also be able to communicate with more than one human entity that is an Emergency Vehicle Driver, each of which may belong to the same Emergency Service, or to different Emergency Services.<br/><br/>Mnemonic: d.evd<br/>
