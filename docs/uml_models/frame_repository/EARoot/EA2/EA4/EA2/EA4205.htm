This Terminator shall represent organisations that are capable of carrying out maintenance activities.  Each organisation shall be represented by one of the following Terminators:<br/><br/>Public Transport Maintenance Organisation<br/>Road Maintenance Organisation<br/><br/>These Terminators shall be capable of both receiving data objects from functionality in the system and providing data objects as input to that functionality.  It shall be possible for the input data objects to be provided both as a response to those received from functionality in the system, or as unsolicited input.<br/><br/>Mnemonic: mo<br/><br/><br/><br/>
