This Terminator is part of the Operator Terminator and shall represent a human entity that is responsible for the management of road bridge(s) that form part of the road network managed by functionality in the system.  It is expected that this Terminator shall provide inputs from which data objects can be created by system functionality to represent commands, data and requests for use by functionality in the system. Each Terminator shall also be capable of translating into human readable form data objects output from the functionality in the system.  The functionality in the system must also be able to communicate with more than one human entity that is a Bridge Operator and for each Bridge Operator to belong to one or separate organisations.<br/><br/>Mnemonic: o.bo<br/>
