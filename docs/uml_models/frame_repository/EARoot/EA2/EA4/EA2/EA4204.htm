This Terminator shall represent external entities that provide data objects to functionality in various parts of the system from which they can determine their location.  Typically these data objects shall be provided continuously without requests being made by any of the functionality and be used to determine the location of any type pf Vehicle and Travellers.  The actual identity of the source of the data is not important, but it could for example be a Global Navigation Satellite Systems (GNSS) such as the European Galileo System or the American Global Positioning System (GPS), or it could be a ground based unit such as a beacon of the type that is used to enable Public Transport Vehicles to determine their location as they follow a proscribed route.<br/><br/>Mnemonic: lds<br/>
