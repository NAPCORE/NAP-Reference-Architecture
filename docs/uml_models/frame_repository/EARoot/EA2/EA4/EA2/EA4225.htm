This Terminator is part of the Driver Terminator and shall represent a human entity that drives a vehicle that is licensed to carry passengers who pay for their transportation and which may or may not use fixed or pre-defined routes.  It is expected that the Terminator shall provide inputs from which data objects can be created by system functionality to represent requests and data for use in the system and be capable of translating into human readable form data objects output from the functionality in the system.  The Vehicle type for this Driver is defined in the description of the Public Transport Vehicle Terminator.  The functionality in the system must also be able to communicate with more than one human entity that is a Public Transport Driver, each of which may belong to the same PT organisation, or to different organisations.<br/><br/>Mnemonic: d.ptd<br/>
