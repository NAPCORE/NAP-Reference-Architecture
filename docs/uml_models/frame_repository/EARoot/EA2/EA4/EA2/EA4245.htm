This Terminator is part of the Traveller Terminator represents a human entity that is currently using a Public Transport (PT) Vehicle to complete some or their entire trip.  It is expected that the Terminator shall provide inputs from which data objects can be generated by functionality in the system to represent requests and data for use by in the system.  The Terminator is also expected to be capable of translating into human readable form the data objects output from the functionality in the system and responding to that output if required.<br/><br/>Mnemonic: t.ptp<br/>
