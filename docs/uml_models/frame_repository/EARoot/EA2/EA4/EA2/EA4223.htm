This Terminator is part of the Driver Terminator and shall represent a human entity that drives a vehicle that is carrying hazardous goods, i.e. a Hazardous Goods Vehicle.  It shall be expected that the Terminator shall provide inputs from which data objects can be created by system functionality to represent requests and data for use in the system and be capable of translating into human readable form data objects output from the functionality in the system.  The vehicle type for this Driver is defined in the description of the Hazardous Goods Vehicle Terminator.  The functionality in the system must also be able to communicate with more than one human entity that drives vehicles carrying hazardous goods, each of which may belong to the same organisation, or to different organisations.<br/><br/>Mnemonic: d.hgvd<br/>
