This Terminator shall represent those systems that detect and monitor weather conditions in the geographic area served by the system.  It is expected that this Terminator shall provide data objects that it has created from its own sensors which shall enable the functionality in the system to react to current and forecast weather conditions so that the flow of traffic in the road network suffers the minimum of disruption and, if necessary. PT services are adjusted to take account of any adverse weather conditions.<br/><br/>Mnemonic: ws<br/>
