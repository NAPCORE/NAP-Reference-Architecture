This Terminator is part of the Multi-modal System Terminator and shall represent the systems that manage the operation of other (non-road based) modes of transport, such as railways, canals, ferry services and airlines.  It shall be possible for the functionality in the system to send data objects to this Terminator requesting either information about the services that are available either now or in the future, or the current status of the services.  The Terminator is expected to respond with data objects containing the requested information for use by functionality in the system.<br/><br/>Mnemonic: mms.mmms<br/>
