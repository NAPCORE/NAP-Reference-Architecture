This Terminator is part of the Vehicle Terminator and shall represent the Other Vehicle.  It shall be part of the normal traffic flow, but may or may not be travelling in the same direction and/or on the same carriageway as the Host Vehicle, or part of the same platoon as the Host Vehicle.  The functionality in the system identifies the Vehicle in which is it operating as the Host Vehicle.  It is expected that this Terminator shall provide data objects to the functionality in the system either directly through an interface or through sensors, and these data objects shall be used to assist in providing the ITS Services.  In addition, the Terminator shall be sent data objects through an interface by functionality in the system that requests the systems in the Host Vehicle to take some action.  Note that this shall be a "request" and not a command, as it shall not be possible for the Host Vehicle to be controlled by the functionality in the system.<br/><br/>Mnemonic: v.ov<br/>
