This Terminator shall represent the human entity that operates any type of licensed Vehicle using any part of the road network.  It shall consist of the following Terminators, each of which shall represent the human entity that drives a particular type of Vehicle:<br/><br/>Emergency Vehicle Driver<br/>Freight Vehicle Driver<br/>Hazardous Goods Vehicle Driver <br/>On-Demand Service Driver<br/>Public Transport Driver<br/>Trip Planning Driver<br/><br/>It is expected that each Terminator and the Driver Terminator itself shall provide inputs from which data objects can be generated by functionality in the system to represent requests and data for use by in the system. Each Terminator and the Driver Terminator shall also be expected to be capable of translating into human readable form data objects output from the functionality in the system and responding to that output if required.<br/><br/>Mnemonic: d<br/>
