This Terminator is part of the Vehicle Terminator and shall represent a Vehicle that delivers personal transport services on-demand.  Usually the Vehicle shall be licensed to carry a small number of passengers as this type of service is not expected to attract a large number of passengers due to its non-scheduled operation and, possibly, variable routes.  It is expected that this Terminator shall provide data objects to the functionality in the system either directly through an interface or through sensors, and these shall be used to assist the provision of On-Demand PT services.  The functionality in the system must also be able to communicate with more than one On-Demand Service Vehicle, each of which may belong to the same organisation, or to different organisations.<br/><br/>Mnemonic: v.odsv<br/>
