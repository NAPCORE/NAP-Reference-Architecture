This Terminator is part of the Other Related System Terminator and shall represent another instance of the functionality in the system that is for use at a Public Transport (PT) Stop.  It shall enable the data objects to be exchanged between PT Stops so that if necessary information can be passed along PT routes.  It shall be possible for the data objects to be sent and received on request, at regular intervals, or because the sending functionality has made changes.<br/><br/>Mnemonic: ors.pts<br/>
