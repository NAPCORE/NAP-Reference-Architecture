This Terminator is part of the Traveller Terminator represents a human entity that is using the functionality in the system to prepare a trip plan to be used at some point in the future.  It is expected that the Terminator shall provide inputs from which data objects can be generated by functionality in the system to represent requests and data for use by the system.  The Terminator is also expected to be capable of translating in human readable form data objects output from the functionality in the system and responding to that output if required.<br/><br/>Mnemonic: t.ptt<br/>
