This Terminator is part of the Operator Terminator and shall represent a human entity that uses the functionality in the system to manage the operation and use of car parks for the organisation that owns and/or operates car park(s).  It is expected that the Terminator shall provide inputs from which data objects can be created by system functionality to represent commands, data and requests for use in the system and be capable of translating into human readable form data objects output from the functionality in the system.  The functionality in the system must be able to communicate with more than one human entity that is a Parking Operator, each of which may belong to the same car park owning/operating organisation, or to different organisations.<br/><br/>Mnemonic: o.po<br/>
