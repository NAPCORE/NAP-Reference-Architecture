This Terminator shall represent the human entity that is responsible for deciding how the traffic using the road network is to be managed.  It is expected that the Terminator shall provide inputs from which data objects can be created by functionality in the system to represent commands, data and requests for use in the system and be capable of translating into human readable form data objects output from the functionality in the system.<br/><br/>Mnemonic: tp<br/>
