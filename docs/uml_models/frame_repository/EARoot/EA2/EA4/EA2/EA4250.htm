This Terminator is part of the Freight Transport Terminator and shall represent a human entity that wishes to transport one or more items of freight (called Freight Transport Load Items) using the available Freight Transport Services.  It shall be possible for the Freight Transport User to request the Freight Transport Operations Manager to prepare a Freight Transport Execution Plan the implementation of which shall enable the Freight Transport Load Items to be moved from one location to another as specified by the Freight Transport User.  When a Freight Transport Execution Plan has been prepared, the Freight Transport User shall have to approve the content of the Plan before it can be implemented by the system.  The Freight Transport User is able to request updates about the progress with the implementation of the Freight Transport Execution Plan.<br/><br/>Mnemonic: ft.ftu<br/>
