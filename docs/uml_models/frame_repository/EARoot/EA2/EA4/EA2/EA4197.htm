This Terminator shall represent all the entities that need to be involved in the movement of freight.  It shall consist of the following Terminators:<br/><br/>Freight Terminal Area Facilities<br/>Freight Transport User<br/>Freight Transport Operations Manager<br/><br/>It is expected that each Terminator shall provide inputs from which data objects can be generated by functionality in the system to represent requests for the movement of freight and the responses to those requests, plus requests and responses concerning the management of the actual freight movement.  Each of the above Terminators shall be expected to be capable of translating into human readable form data objects output from the functionality in the system and responding to that output if required.<br/><br/>Mnemonic: ft<br/>
