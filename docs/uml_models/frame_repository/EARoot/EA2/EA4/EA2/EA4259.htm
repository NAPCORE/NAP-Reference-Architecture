This Terminator is part of the External Service Provider Terminator and shall represent an organisation that provides a service through which Travellers can obtain traffic and travel information for the road network. It is expected that this Terminator shall at various times input data objects containing requests for traffic and travel information because incidents have occurred, to which the incident management functionality shall respond.  Without previous requests, other functionality in the system may also at various times provide this Terminator with data objects containing both current and forecast traffic and travel information, plus information about weather conditions.<br/><br/>Mnemonic: esp.ttip<br/>
