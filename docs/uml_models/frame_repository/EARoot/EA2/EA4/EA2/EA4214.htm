This Terminator shall represent the roles of a human entity that uses (or is about to use) ITS Services provided by functionality in the system.  It shall be possible for the human entity this Terminator represents to switch between several roles during their use the ITS Services provided by the functionality in the system.  Each role shall be represented by one of the following Terminators:<br/><br/>Cyclist<br/>On-Demand Service Passenger<br/>Pedestrian<br/>Pre-Trip Traveller<br/>Public Transport Passenger<br/>Static Traveller<br/>Vehicle Sharer<br/><br/>It is expected that each Terminator shall provide inputs from which data objects can be generated by functionality in the system to represent requests and data for use by in the system.  Each Terminator is also expected to be capable of translating into human readable form the data objects output from the functionality in the system and responding to that output if required.<br/><br/>Mnemonic: t<br/>
