This interface is used to provide metadata to the NAP.<br/><br/>As a guideline for minimum expectation the following values can be used: <br/><br/>Data type: Raw data<br/><br/>max bytes per message<br/>up to 500 kBytes for meta data<br/>up to 10 MBytes for attached files<br/><br/>maximum delay [s]<br/>system response time below 100ms (independent from user internet connection)  <br/>delay for uploading meta data: 0.5s<br/>delay for uploading attached files: 10s<br/><br/>message interval<br/>one minute - assumption that two data provider are putting data in parallel at the same time; depends on client numbers<br/><br/>security level<br/>level high:<br/>authorization by the system<br/>encrypted data transfer on system side<br/><br/>transfer mode<br/>wired connection<br/>
