It contains data objects created by Provide Metadata Quality Checks after checking quality of metadata that has been retrieved from the store of NAP Metadata following a scheduled or triggered event. The data is timestamp and success of failure (with number of errors found, level of service achieved) of the checks and overall aggregated summary.
