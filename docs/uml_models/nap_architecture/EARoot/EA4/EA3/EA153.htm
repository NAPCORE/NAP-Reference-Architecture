It contains outputs to the NAP Content Consumer that have been created from data objects provided by functionality in the Manage Traffic Functional Area that are the response (“acceptance” or “rejection”) to a previous registration request from a NAP Content Consumer. In case of a successful registration, it includes credentials for the Content Consumer to prove its authorisation to access NAP functionality that needs authorisation.
