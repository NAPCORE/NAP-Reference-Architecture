This Function shall be capable of providing the following facilities:<br/><br/>(1) The ability to provide periodical and initial checking of published data quality based on prespecified rules and procedures (including appropriate time frequencies). Main tests performed are schema validation and availability of provided data against the metadata information. <br/>(2) The ability to retrieve data to be checked from the NAP Data store.<br/>(3) The ability to disseminate and store the outcomes of data quality checks to the NAP Data store.<br/>(4) The function shall be capable to request  metadata from NAP Metadata store, by invoking the functionality of Manage NAP Data Repository  for data validation purposes <br/>(5) Ability to perform the checks based on data events (data insert, data update) in the NAP Data store<br/><br/>==========<br/><b>CHANGELOG:</b> <br/>- change name from “Provide Data Quality Checks” to “F3.6.10 Provide Data Quality Checks “  <br/>- first proposal of the text <br/>- added ability to request metadata from NAP Metadata store <br/>
