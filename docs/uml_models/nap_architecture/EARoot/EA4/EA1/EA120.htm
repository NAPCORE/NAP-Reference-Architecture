This application represents an application that handles User Registrations (i.e. NAP Content Providers and NAP Content Consumers).<br/>It shall contain a NAP Users data store and an interface to  Manage NAP User Registrations and Provide Authorisation Service. It shall also be able to verify NAP Content Providers to authorise the usage of the NAP Metadata store. <br/><br/>This generic NAP User Management Component represents a component of the NAP that handles user registrations, logins and authorisations using the User Management application.<br/><br/><br/><br/>===========<br/><b>CHANGELOG:</b><br/>- change of APP name to capitals<br/>- total rewrite of the capability<br/>- update of names of functions in the description<br/><b>TODO:</b><br/>- decide if applications are necessary.<br/><br/>
