The generic terminator role of the NAP Content Consumer represents any interested entity in traffic and transport data. The Content Consumer can find available data via the NAP. Therefore, Function F3.6.4 Provide Access for NAP Meta-Data Retrieval offers an interface (typically a graphical user interface freely accessible via the Internet) to search the meta-data in D3.15 NAP Meta-Data<br/><br/><b>Key issue</b>: Request high-quality services; provide feedback about relevant content<br/><b>Area of responsibility</b>: Be compliant with the agreement with the NAP content provider<br/><b>Tasks</b>: Receive content <br/><b>Mnemonic</b>: nap.cc<br/>
