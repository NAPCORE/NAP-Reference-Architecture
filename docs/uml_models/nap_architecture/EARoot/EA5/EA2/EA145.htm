The generic terminator role Metadata Requesting System represents any interested entity in traffic and transport metadata dissemination. The Metadata Requesting System can request information stored in the NAP Metadata store in a standardized format (mobilityDCAT-AP) to incorporate said data into its information services (i.e. metadata harvesting). Therefore, Function F3.6.4 Manage NAP Metadata Retrieval offers a data distribution interface for machine-to-machine data retrieval.  <br/><br/><b>Key issue</b>: Request metadata stored at NAP.  <br/><b>Area of responsibility</b>: Be compliant with the agreement with the NAP operator  <br/><b>Tasks</b>: Receive information about the content in a standardized manner, i.e. metadata   <br/><b>Mnemonic</b>: nap.mdrs  <br/>
