It consists of inputs from the NAP Content Consumer containing data objects that provide request for retrieval of traffic or travel data stored in NAP Data store. Besides the identification of data to be retrieved the request must contain the authorisation credentials of the entity that is providing it, so that this can be checked to ensure that it is a registered entity with right to access the data.  <br/><br/>NOTE: authorisation credentials might be void if the data to be retrieved are freely available (i.e. open data)   <br/>
