The Data Platform NAP sub system represents the hardware/software implementation of the ITS Service. It consists of modules performing data and metadata quality checks, hosting data and metadata, user data, and credentials, allowing the creation and retrieval of metadata records as well as real data from the internal database and user support. <br/>
