<b>Type</b>: Use interface <br/><b>Short description</b>: This interface is used for the interaction of the user management and metadata repository.<br/><b>Contextual Use</b>: It is assumed that the NAP shall have a registration procedure for advanced content consumer. The registration can differ in Member States. The trust of the registered person should be guaranteed and should follow international standards. To enable machine to machine meta data harvesting a registration of such a interface and credentials is needed. <br/><b>Related Documents</b>: https://napcore.eu/activity-wg2-interoperability-and-level-of-service-of-naps/ <br/>=== Communications Requirements ===<br/>As a guideline for minimum expectation the following values can be used<br/><b>Data type</b>: <br/><ul>
<li>Raw data</li></ul>
<b>max bytes per message</b><br/><ul>
<li>up to 500 kBytes for meta data</li></ul>
<b>maximum delay [s]</b><br/><ul>
<li>system response time below 100ms (independent from user internet connection)  </li></ul>
<b>message interval</b><br/><ul>
<li>one second - assumption that 1 system request metadata</li></ul>
