<b>Title</b>: Transmodel<br/><b>Reference</b>: EN 12896<br/><b>Type</b>: Technical Specification<br/><b>Full Name and Hyperlink</b>: Transmodel - Public Transport Reference Data Model<br/><b>Contextual Use</b>: Transmodel provides a comprehensive conceptual model for public transport information systems. It covers multiple subdomains including transport network infrastructure and topology, schedules, journey planning, fares, fare validation, real-time passenger information, and operational systems. The model facilitates interoperability between different public transport information systems and supports the integration of various transport modes.<br/><b>Document Number</b>: EN 12896<br/><b>Publication Date</b>: Ongoing development (latest version referenced)<br/><b>Related Documents</b>:<br/><ul>
<li><b>NeTEx (Network Timetable Exchange)</b>: A standard for exchanging public transport schedules and related data.</li><li><b>SIRI (Service Interface for Real Time Information)</b>: A standard for exchanging real-time public transport information.</li><li><b>IFOPT (Identification of Fixed Objects in Public Transport)</b>: A standard for identifying fixed objects such as stops and stations in public transport.</li><li><b>CEN TS 16614</b>: Technical Specification for NeTEx.</li><li><b>CEN TS 15531</b>: Technical Specification for SIRI.</li></ul><p/>
