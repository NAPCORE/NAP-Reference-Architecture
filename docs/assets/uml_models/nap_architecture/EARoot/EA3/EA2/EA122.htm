The generic terminator role Data Requesting System represents any interested entity in traffic, travel data. The Data Requesting System can request traffic and travel data available in the NAP Data store. Therefore, Function F3.6.9 Manage NAP Data Retrieval offers a data distribution interface for machine-to-machine data retrieval.   <br/>In some cases, especially when the data are pushed to the Content Consumer systems and / or authorisation is necessary the Data Requesting System must use credentials obtained by Content Consumer. When registering the Content Consumer will obtain credentials and / or to set up a machine account to which data are pushed.    <br/><br/><b>Key issue</b>: Request data stored at NAP.  <br/><b>Area of responsibility</b>: Be compliant with the agreement with content provider  <br/><b>Tasks</b>: Receive content   <br/><b>Mnemonic</b>: nap.drs  <br/>
