This Data Store shall be used within the Manage Traffic Functional Area.  It shall contain information objects that provide traffic and travel data.   <br/><br/>Each information object in the Store shall contain the following data objects: <br/><br/>(1) actual stored traffic or travel related data that is considered as a compact and comprehensive data content <br/>(2) data quality information  <br/>(3) relation of the data content to the metadata elements, allowing to identify data distribution <br/>(4) access credentials to stored content identifying the information object creator and updater and its readers <br/>(5) last update time information <br/>(6) size of the object <br/><br/>Data content to be stored in the data store is given mainly by the obligations to make certain data content accessible via the NAP. The obligation together with a list of data is stipulated in the ITS Directive 2010/40/EU and its revised version (EU) 2023/2661 and complementary Delegated Regulations 885/2013, 886/2013, 2015/962, 2017/1926, 2022/670 and 2023/1804.<br/><br/>==========  <br/><br/>TODO: <br/>- check if information object is complete <br/>
