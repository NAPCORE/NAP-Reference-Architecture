(a) whenever the illegal use data flow is received indicating that the vehicle is being used illegally, send the stolen vehicle notification data flow to functionality in the Provide Safety and Emergency Facilities Functional Area and include in it the vehicle ID and vehicle location;<br/>(b) when as a result of (a) the send stop message data flow is received, send the stop stolen vehicle data flow to the Communicate With In-vehicle Systems function;<br/>(c) when after completing (a) the vehicle position for illegal use data flow is received, check if the Vehicle has moved and if so, resend the stolen vehicle notification data flow to functionality in the Provide Safety and Emergency Facilities Functional Area and include in it the new Vehicle location;<br/>(d) when the vehicle ID for illegal use or vehicle position for illegal use data flows are received retain their contents for use when the vehicle is being used illegally.<br/>
