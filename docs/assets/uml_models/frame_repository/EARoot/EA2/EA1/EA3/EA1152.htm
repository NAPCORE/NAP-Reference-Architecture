This High-level Function provides facilities for the prediction of traffic conditions using simulation and modelling techniques.  It will provide individual facilities for the creation of traffic management strategies through the use of simulation tools, the use of data objects containing real-time data to drive the simulation tools, the creation and use of models of the road networks and the recording of which strategies have been used and the results of their use so that their effect on the traffic conditions in the road network can be determined and assessed, i.e. learning the lessons from previous strategy implementations.  Many of these facilities are provided as options so that it will be possible to configure individual systems to contain some or all of them.  The Road Network Operator will be able to monitor and manage these facilities and hence the use of the simulation and modelling tools.<br/>
