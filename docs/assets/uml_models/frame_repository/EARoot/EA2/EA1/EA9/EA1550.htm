Appears In: DFD F9.3 Manage Vehicle access to Sensitive Areas, Functional Tree of Area F9 Provide Support for Cooperative Systems
This Function shall be capable of providing the following facilities:<br/><br/>(1) A HMI through which the Road Network Operator can manage the criteria that are used to control the access of Vehicles to "sensitive areas" within the road network.<br/>(2) The ability for the Operator to use the HMI to request details of the current criteria and to update them.<br/>(3) The ability for the Operator to request and be provided with a report on the use that Vehicles have made of the "sensitive areas" in the road network.<br/>

Details:
(a) continuously monitor for receipt of the request current vehicle access criteria, update current vehicle access criteria and request sensitive area vehicle use report data flows;<br/>(b) when the first data flow in (a) is received, put the request in the data flow request current vehicle access data and send it to the Manage Data about Vehicle user of Sensitive Areas function;<br/>(c) as a result of (b) continuously monitor for receipt of the data flow current vehicle access data;<br/>(d) when the data flow in (c) is received, output its contents to the Road Network Operator in the current vehicle access criteria data flow;<br/>(e) when the second data flow in (a) is received, send the updated criteria to the Manage Data about Vehicle use of Sensitive Areas function in the update current vehicle access criteria data flow;<br/>(f) when the third data flow in (a) is received, put the request in the data flow request current vehicle access data and send it to the Manage Data about Vehicle user of Sensitive Areas function;<br/>(g) as a result of (b) continuously monitor for receipt of the data flow current vehicle access data;<br/>(h) when the data flow in (g) is received, output its contents to the Road Network Operator in the requested sensitive area vehicle use report data flow.<br/>
