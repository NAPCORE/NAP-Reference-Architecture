This Function shall be capable of providing the following facilities:<br/><br/>(1) The ability to receive data objects about the road network covering both static data (e.g. road layout and geometry) and dynamic data (e.g. historic, current and predicted traffic flows) from functionality in other parts of the system.<br/>(2) The ability to receive data objects containing weather related information for the geographic area covered by the road network.<br/>(3) The ability to receive data objects containing the previous performance of the implementation of Freight Transport Execution Plans.<br/>(4)The ability to manage the store of Road Network Data, including the integration of the data objects received in (1), (2) and (3) above.<br/>(5) The ability to respond to requests from other functionality in the system for some or all of the content of the store of Road Network Data. <br/><br/>

Details:
(a) continuously monitor for receipt of any data flows that have been sent by functionality in the Manage Traffic Functional Area (F3), the data flow request road network data from the Collect FTEP Data from FTOM and MMS function and the deduced road network performance data flow from the Evaluate FTEP Performance function;<br/>(b) when any of the data flows from the functionality in the Manage Traffic Functional Area are received extract their content and load it into the appropriate place(s) in the Road Network Data Store;<br/>(c) when the deduced road network performance data flow from the Evaluate FTEP Performance function is received, extract its content and load it into the appropriate place(s) in the Road Network Data Store;<br/>(d) when the request road network data from the Collect FTEP Data from FTOM and MMS function is received, extract the road network data for the specified geographic areas from the Road Network Data Store and send it back to the Collect FTEP Data from FTOM and MMS function in the data flow requested road network data.<br/>
