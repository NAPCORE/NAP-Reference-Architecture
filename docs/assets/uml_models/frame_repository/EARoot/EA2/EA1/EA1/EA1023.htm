Appears In: Functional Tree of Area F1 Provide Electronic Payment Facilities, DFD F1.5 Access and Credit Control
This Function shall be capable of providing the following facilities:<br/><br/>(1) The ability to control the access of a user (Traveller or Driver) to the requested service.<br/>(2) Upon detection of an incorrect payment (fraud), the ability to send a message to the Detect Access Violations function.<br/>(3) If an accident has been detected, the ability to send a warning to functionality in the Manage Traffic Functional Area.<br/>(4) The ability to let the user proceed upon receipt of an agreement message from the Detect Access Violations function.<br/>(5) A HMI through which the user can be informed about whether or not access to the requested service has been granted.<br/>
