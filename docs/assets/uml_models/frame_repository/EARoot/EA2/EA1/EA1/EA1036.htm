This Function shall be capable of providing the following facilities:<br/><br/>(1) A HMI through which the Traveller can establish a contract with the provider of ITS Services to be used during the implementation of a trip plan.<br/>(2) The HMI shall present the Traveller with information about the different types of ITS Services available, and record the choices that they make for use by other functionality.<br/>

Details:
(a) continuously monitor for receipt of the service contract information data flow from the Plan Trip Details function in the Provide Traveller Journey Assistance Functional Area;<br/>(b) when the data flow in (b) is received use the request service data data flow to request from the Manage Services Data function a copy of the information about the ITS Services that have been received;<br/>(c) when the available service data data flow is received as a result of (b), send the contents to the Traveller in the service information data flow;<br/>(d) continuously monitor for receipt of the contract data data flow from the Traveller;<br/>(e) when the data flow in (c) is received, check the context against the data objects received in (b);<br/>(f) when a match is found in (d), write the contract;<br/>(g) display the contract to the Traveller in the contract data flow;<br/>(h) send a copy of the contract to the store of EP Contract Data in the load contract details data flow;<br/>(i) if no match is found in (d), send the message no contract available to the Traveller in the contract data flow.<br/>
