This Function shall be capable of providing the following facilities:<br/><br/>(1) An interface that enables for the Operators and Service Providers to view all the transactions concerning the services that they provide.<br/>(2) The ability for the interface to only provide information to Operators and External Service Providers on request.<br/>(3) The interface shall only have the ability to provide information to External Service Providers that is relevant to them and not information that is relevant to other Service Providers.<br/>
