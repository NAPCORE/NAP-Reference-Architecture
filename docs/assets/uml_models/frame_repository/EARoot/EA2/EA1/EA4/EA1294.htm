This Function shall be capable of providing the following facilities:<br/><br/>(1) The ability to calculate the performance for the operation of the PT Services.<br/>(2) The ability for the performance calculations to be based on continuously observed real time data collected from the operating PT Vehicles by other functionality.<br/>(3) The ability for the performance calculations to be initiated by the PT Operator through the PT Operator Interface Function, to which the results shall be automatically sent. <br/>(4) The ability to send the results of the performance calculations to the Optimise Control Action Function.<br/>
