Appears In: DFD F4.3 Provide PT Management, Functional Tree of Area F4 Manage Public Transport Operations
This Function shall be capable of providing the following facilities:<br/><br/>(1) A HMI through which the PT Operator can manage the creation of schedules for PT Drivers.<br/>(2) The ability to use this HMI to update the details about individual PT Drivers, such as age, ability, experience, availability, etc. after which a new schedule for the Driver's work will be created.<br/>(3) The ability to use the HMI to request and have output the current details about the PT Driver, including the current work schedule.<br/>(4) The ability to use the HMI to be provided with the reason(s) why the creation of a new PT Driver schedule has failed.<br/>
