It contains data objects that provide details of all the violation committed by a given user during a certain period.  The data flow contains the following data objects:<br/>- user ID,<br/>- period concerned,<br/>- list of violations committed, with:<br/>       - type of violation,<br/>       - date of violation,<br/>       - location of violation,<br/>       - violation notification and prosecution files references.<br/>
