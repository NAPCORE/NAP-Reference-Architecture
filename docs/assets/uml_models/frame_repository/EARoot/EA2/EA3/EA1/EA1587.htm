It contains inputs that a Driver has made which are used to create data objects that provide payment of the fee required for the use of a Service and is for use by functionality in the Provide Electronic Payment Facilities Functional Area.  It is assumed that the payment is made in some form of electronic transaction, i.e. it is not cash.<br/>
