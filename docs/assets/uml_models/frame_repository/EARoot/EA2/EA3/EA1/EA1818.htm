It contains data objects that provide the structure of the road network.  This includes information for trip planning for all kinds of vehicles: restrictions for certain freight (e.g. hazardous goods) or vehicles (e.g. weight, dimension, etc.), access restrictions (e.g. one direction), but also travel times between locations in the network.  Extra access rights for emergency vehicles (e.g. ambulance) are included.<br/>
