Appears In: DFD F7.5 Manage Rules and Users' Registrations
It contains data objects that provide details of the owner or person responsible for a Vehicle and what the vehicle or user is allowed to do within the road transport network that are being loaded into the store of User's Registration Data.  Each time the data flow is used it will contain some or all of the following data objects:<br/>- user ID,<br/>- Vehicle ID,<br/>- ID's of other Vehicles used by the user,<br/>- operations allowed for the Vehicle,<br/>- period of validity of registration,<br/>- max speed according to environment conditions<br/>- max weight (total and per axle)<br/>- max pollution level <br/>- type of cargo / types of forbidden zones<br/>- max duration that a Driver can operate a Vehicle without a break.<br/>
