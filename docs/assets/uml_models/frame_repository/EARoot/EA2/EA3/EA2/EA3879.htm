It contains output to the Freight Vehicle Driver that has been created from data objects provided by functionality in the Provide Support for Host Vehicle Systems Functional Area and includes the response to the previous request for a parking space at a rest to be booked.  The response may be either acceptance of the previously requested booking, or details of alternative times when a parking space is available, because it is not available at the requested time.<br/>
