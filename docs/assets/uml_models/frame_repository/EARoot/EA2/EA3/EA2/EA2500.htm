It contains data objects that provide details of a route the preparation of whch by functionality in the Provide Support for Host Vehicle Systems Functional Area is requested for inclusion in a Freight Transport Execution Plan.  It consists of some or all of the following data objects:<br/>- the date the implementation of the Freight Transport Execution Plan is to commence,<br/>- departure time from the road based Freight Transport Execution Plan origin,<br/>- the Freight Transport Execution Plan origin location,<br/>- the date Freight Transport Execution Plan is to complete (may be same as commence date),<br/>- required arrival time at the road based Freight Transport Execution Plan destination,<br/>- the Freight Transport Execution Plan destination location,<br/>- if the Freight Transport Execution Plan destination is a Freight Terminal Area, the facilities required,<br/>- number of Freight Transport Items to be moved by the Freight Transport Execution Plan,<br/>- type of Freight Vehicle required.<br/>
