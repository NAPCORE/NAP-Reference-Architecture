It contains data objects that provide the current status of an active eCall message.  See description of D2.4 for details of the contents of this data flow.<br/>Note that if the only data object that actually contains data is the identity of the Vehicle (VIN) then this will mean that the eCall data for this Vehicle is to be deleted because the eCall is no longer active.  If the VIN is "XXX" then the content of the Data Store is to be provided.<br/>
