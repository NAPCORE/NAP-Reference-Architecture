It contains inputs the Freight Transport User has made from which data objects are created to provide a request for a new Freight Transport Execution Plan to be created using the supplied data objects.  As a minimum these data objects shall include but not be limited to the following:<br/>- the number of Freight Transport Load Item(s) to be moved;<br/>- the identity of each Freight Transport Load Item,<br/>- the description of each Freight Transport Load Item including any special features, e.g. hazardous materials and fragile,<br/>- the origin, destination, time of departure and expected time of arrival (eta) of each Freight Transport Load Item.<br/>At the discretion of the Freight Transport User, data objects may also be included to specify the following:<br/>- a particular route to be followed,<br/>- the modes of transport that can be employed,<br/>- the existence of a Freight Transport Execution Plan that was used to previously move similar Freight Transport Load Items.<br/>
