It contains data objects that provide a request from the Freight Vehicle Driver via part of the Vehicle Trip Planning Interface for the use of a loading or unloading zone.  Included is the desired location, date/time, duration for which the zone is expected to be occupied, Vehicle type and type of goods, including type of Hazardous Goods if relevant.<br/>
