It contains data objects that describe a fully prepared Freight Transport Execution Plan with details of the Freight Transport Load Item(s) that are to be carried, their origin(s), itinerary, their destination(s) and the resources required to implement the Plan which constitute the implementation instructions for the FTEP.  The data objects included in this data flow are those shown in the definition of the FTEP Data Store, but some may be blank (null) if not required, e.g. if the use of other non-road based transport modes is not needed then the relevant data objects will be blank.<br/>
