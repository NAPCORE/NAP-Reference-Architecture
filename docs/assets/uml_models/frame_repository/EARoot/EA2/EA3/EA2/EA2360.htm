It consists of input from the Traveller from which data objects are created that provide details that enable the Traveller to pay for the use of a Service and are for use by functionality in the Provide Electronic Payment Facilities Functional Area.  The inputs include: selected mode of payment (e.g. debit the Traveller's EP account, or a specified type of card, or electronic payment using an app), plus if using the Traveller's Electronic Payment (EP) account: the account ID and the selected mode of debiting (immediate / differed / scheduled).<br/>
