It contains outputs to the Traveller that have been created from data objects provided by functionality in the Provide Electronic Payment Facilities Functional Area and includes details about the transaction performed by the Traveller during a certain period.  The data objects include the following elements:<br/>- user ID or vehicle ID<br/>- for each transaction :<br/>      - date / time<br/>      - location<br/>      - operator or information provider ID<br/>      - service ID<br/>      - parameters allowing to precisely define the service used<br/>      - ID of contract used<br/>      - account ID<br/>      - corresponding amount<br/>
