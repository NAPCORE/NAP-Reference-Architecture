It contains data objects that are a requirement for a "free ride" to be given to a Vehicle using the inter-urban road network, e.g. Emergency Vehicle, or other form(s) of special Vehicle.  Thus when it passes through toll collection points there will be no payment control . The data flow includes the following data objects: <br/>- date of message,<br/>- ID of emitter,<br/>- ID of user or vehicle to be allowed to pass freely,<br/>- date of validity,<br/>- part(s) of the inter-urban road network to which free use applies,<br/>- other relevant information.<br/>
