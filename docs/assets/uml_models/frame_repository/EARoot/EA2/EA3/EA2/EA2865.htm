It contains data objects that provide information about unusual traffic conditions within the inter-urban road network that are for use in monitoring a trip being implemented by a Traveller using a nomadic device.  The causes of these conditions may include such things as road works and road blocks.  In both cases the current travel times may exceed normal times.<br/>
