It contains outputs to the Financial Clearing House that have been created from data objects provided by functionality in the Provide Electronic Payment Facilities Functional Area and includes a request to transfer money to an account of an operator (or information provider).  The information objects include, date, financial clearing-house ID, operator or information provider ID, ID of the account to be credited, ID of the EP account and amount to be transferred.<br/>
