It contains data objects that provide all the elements making it possible to establish the prosecution file.  The data is based on the violation notification, to which are added the following:<br/>- data of violation,<br/>- location at which violation tool place,<br/>- any images or records of the violation,<br/>- classification of violation (seriousness and type),<br/>- description of the violation, e.g. rule that was broken,<br/>- Vehicle ID (if applicable),<br/>- involved user ID,<br/>- location at which violation tool place<br/>- other offences references enacted by the same person,<br/>- violation message output to roadside.<br/>
