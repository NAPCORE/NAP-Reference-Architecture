Appears In: DFD F5.13 Provide Vehicle Data and ISA
This Data Store shall be used within the Provide Support for Host Vehicle Services Functional Area.  It shall contain information objects that provide speed limit data for a region from which, given a set of co-ordinates, the speed limit for that section of road can be identified.<br/><br/>The information object in the Store shall contain the following data objects:<br/>(1) Road link ID (number),<br/>(2) Location (characters and/or numbers, e.g. GPS/Galileo data),<br/>(3) Speed limit (number in kph or mph),<br/>(4) Recommended speed (number in kph or mph),<br/>(5) Weather conditions for recommended speed (characters).<br/>
