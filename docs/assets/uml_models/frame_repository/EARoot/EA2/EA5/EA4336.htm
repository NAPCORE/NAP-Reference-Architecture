This Data Store shall contain information objects that contain the predefined responses to incidents.  These responses shall have been created by the Freight Transport Operations Manager.  There shall be one information object for each incident response and each information object consists of, but is not limited to, the following data objects:<br/>(1) incident response number (number),<br/>(2) date of creation (number),<br/>(3) time of creation (number),<br/>(4) creator identity or name (characters),<br/>(5) number of times used (number),<br/>(6) data last modified (number),<br/>(7) time last modified (number),<br/>(8) modifier identity of name (characters),<br/>(9) type of incident covered by the response (characters),<br/>(10) generic location description, e.g. on the road, in a parking area, or at a Freight Terminal Area (characters),<br/>(11) type of Freight Transport Load Item(s) likely to be affected (characters),<br/>(12) possible effects on Freight Transport Load Items (characters),<br/>(13) required response(s) (characters).<br/>Note tha the required response can include informing the incident management functionality in the Manage Traffic Functional Area (F3).<br/>
