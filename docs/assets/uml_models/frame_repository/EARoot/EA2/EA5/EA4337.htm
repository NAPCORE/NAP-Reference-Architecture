This Data Store shall contain information objects that provide details of the facilities available at Freight Terminal Areas.  Each Freight Terminal Area shall have its own set of information objects, each of which shall describe one facility and contain, but not be limited to, the following data objects:<br/>(1) Freight Terminal Area (FTA) identity (number),<br/>(2) FTA location co-ordinates (number),<br/>(3) date of creation of facility description (number).<br/>(4) time of creation of facility description (number),<br/>(5) description of the FTA facility (characters),<br/>(6) description of access to the FTA facility (characters),<br/>(7) any restrictions on access to the FTA facility (characters).<br/>
