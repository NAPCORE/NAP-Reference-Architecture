Appears In: DFD F2.1.11 Manage eCall Messages
This Data Store shall be used within the Provide Security and Emergency Facilities Functional Area.  It shall contain information objects that provide details of each Public Service Access Point (PSAP) that is able to respond to eCall messages from Vehicles.<br/><br/>Each information object in the store shall contain the following data objects:<br/>(1) PSAP name (character string),<br/>(2) PSAP identity (number - optional),<br/>(3) geographic location of the PSAP (co-ordinates),<br/>(4) geographic area from which the PSAP can respond to eCall messages (co-ordinates),<br/>(5) type(s) of service available from the PSAP,<br/>(6) communication information for the PSAP.<br/>
