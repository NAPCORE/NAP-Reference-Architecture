This Terminator is part of the Freight Transport Terminator and shall represent the systems that provide the facilities which are available at a Freight Terminal Area.  It shall be possible to book the use of these facilities for use by one or more of the Freight Transport Load Items being moved through the implementation of a Freight Transport Execution Plan.  These facilities shall comprise, but not be limited to, the ability to store Freight Transport Load Items either to await collection by another mode of transport, or for collection by a Freight Vehicle for a road based movement.<br/><br/>Mnemonic: ft.fta<br/>
