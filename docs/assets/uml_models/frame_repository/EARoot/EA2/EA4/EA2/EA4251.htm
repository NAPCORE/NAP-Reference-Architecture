This Terminator is part of the External Service Provider Terminator and shall represent a form of External Service Provider that provides information about services such as accommodation, leisure and sports events for which a booking is needed for attendance.  Functionality in the system shall be able to send this Terminator data objects that request particular information be provided.  The data objects received in response by the functionality shall be used in the provision of ITS Services.  Any booking and/or payment for services about which information has been provided are expected to be carried out by functionality not included within the system.<br/><br/>Mnemonic: esp.bsp<br/>
