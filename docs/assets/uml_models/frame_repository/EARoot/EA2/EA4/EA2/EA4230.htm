This Terminator is part of the Operator Terminator and shall represent a human entity that uses the facilities of the system to manage the way that the functionality in the system helps the Emergency Services to respond to incidents.  It is expected that the Terminator shall provide inputs from which data objects can be created by system functionality to represent commands, data and requests for use in the system and be capable of translating into human readable form data objects output from the functionality in the system.  The functionality in the system must also be able to communicate with more than one human entity that is an Emergency Operator, each of which may belong to the same Emergency Service, or to different Emergency Services.<br/><br/>Mnemonic: o.eo<br/>
