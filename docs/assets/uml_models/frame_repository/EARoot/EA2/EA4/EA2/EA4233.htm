This Terminator is part of the Operator Terminator and shall represent a human entity that uses the functionality in the system to manage the provision and operation of Public Transport (PT) services by a Public Transport organisation.  It is expected that the Terminator shall provide inputs from which data objects can be created by system functionality to represent commands, data and requests for use in the system and be capable of translating into human readable form data objects output from the functionality in the system.  The functionality in the system must also be able to communicate with more than one human entity that is a Public Transport Operator, each of which may belong to the same PT organisation, or to different organisations.<br/><br/>Mnemonic: o.pto<br/>
