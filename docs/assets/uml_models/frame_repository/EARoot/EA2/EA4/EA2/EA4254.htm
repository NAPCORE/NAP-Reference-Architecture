This Terminator is part of the External Service Provider Terminator and shall represent an organisation that provides information about places that offer Personal Services (PS) such as garages, shops, banks, post offices, etc. and Places of Interest (POI), such as tourist sites, town and city plans, etc.  It shall be possible for functionality in the system to send this Terminator data objects containing requests for details of PS or POI. The Terminator shall be expected to respond with data objects that contain the requested details for use by the functionality that made the request.<br/><br/>Mnemonic: esp.gip<br/>
