This Terminator is part of the Traveller Terminator and shall represent a human entity that is using the functionality in the system to help them travel through the road network using a bicycle, tricycle, scooter or similar mode of transport.  It is expected that the Terminator shall provide inputs from which data objects shall be generated by functionality in the system to represent requests and data for use by in the system.  The Terminator shall also be expected to be capable of translating into human readable form data objects output from the functionality in the system and responding to that output if required.<br/><br/>Mnemonic: t.c<br/>
