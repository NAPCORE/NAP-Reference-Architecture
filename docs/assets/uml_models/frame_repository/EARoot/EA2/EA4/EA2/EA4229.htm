This Terminator is part of the Operator Terminator and shall represent the human entity that uses the functionality in the system to manage the operation of the electronic payment facility provided by the system.  It is expected that the Terminator shall provide inputs from which data objects can be created by system functionality to represent commands, data and requests for use in the system and be capable of translating into human readable form data objects output from the functionality in the system.  The functionality in the system must also be able to communicate with more than one human entity that is an Electronic Payments Operator, each of which may belong to the same organisation, or to different organisations and may be responsible for one or more electronic payment systems.<br/><br/>Mnemonic: o.epo<br/>
