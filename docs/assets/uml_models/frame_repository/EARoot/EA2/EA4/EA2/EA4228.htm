This Terminator is part of the Operator Terminator and shall represent a human entity that provides and reviews information objects containing communication details for each Public Safety Access Point (PSAP) provider.  The human entity shall be an employee of a Communications Network Provider that is involved in any ITS Service implementation.  It is expected that the Terminator shall provide inputs from which data objects can be created by functionality in the system to represent commands, data and requests, for use in the system.  Each Terminator shall also be capable of translating into human readable form data objects output from the functionality in the system.  The functionality in the system must be able to communicate with more than one human entity that is a Communications Operator, each of which may belong to the same organisation, or to different Communications Network Providers.<br/><br/>Mnemonic: o.co<br/>
