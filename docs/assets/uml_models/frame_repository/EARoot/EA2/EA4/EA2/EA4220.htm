This Terminator is part of the Multi-modal System Terminator and shall represent a system that is responsible for the conveyance of freight using modes of transport other than road, e.g. water, air, and rail.  It shall be possible for functionality in the system to exchange data objects with these systems to enable the synchronisation of the use of the different transport modes in order to maximise the efficiency of freight transport, e.g. to reduce the waiting time at modal interchanges.<br/><br/>Mnemonic: mms.omfs<br/>
