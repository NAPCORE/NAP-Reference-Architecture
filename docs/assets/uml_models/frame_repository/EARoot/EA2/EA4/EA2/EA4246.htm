This Terminator is part of the Traveller Terminator represents a human entity that is using the functionality in the system for a journey but at this moment is not moving, i.e. static, because for example they are waiting at a bus stop, or within a transport interchange.  It is expected that the Terminator shall provide inputs from which data objects can be generated by functionality in the system to represent requests and data for use by the system.  The Terminator is also expected to be capable of translating in human readable form data objects output from the functionality in the system and responding to that output if required.<br/><br/>Mnemonic: t.st<br/>
