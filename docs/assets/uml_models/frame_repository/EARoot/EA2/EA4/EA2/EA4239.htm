This Terminator is part of the Bridge / Tunnel Infrastructure Terminator and shall represent the output provided by sensors that can detect changes in the structure of a bridge or external systems able to provide the same or similar outputs.  These outputs shall be analysed by functionality in the system to create data objects that enable changes in the status of the bridge to be monitored.  In particular if that status reaches a point where it becomes dangerous for vehicles, pedestrians and cyclists to continue to use the bridge the appropriate action can be taken by functionality in the system.  The functionality in the system must also be able to communicate with more than one sensor and/or external system.<br/><br/>Mnemonic: bti.bs<br/>
