It contains outputs to the Financial Clearing House that have been created from data objects provided by functionality in the Provide Electronic Payment Facilities Functional Area and includes a request for details of the current status of a traveller's account that is used to purchase services.  The information objects include, the EP account ID, the financial clearing-house which manages the source from which the money will be transferred and the ID of the source account.<br/>
