It contains data objects that have been created from inputs provided by the Freight Transport User for use in creating a new Freight Transport Execution Plan.  These data objects shall contain the following items of data:<br/>- number of Freight Transport Load Items,<br/>- for each Freight Transport Load Item,<br/>    - description,<br/>    - origin location,<br/>    - required departure date/time,<br/>    - destination location,<br/>    - required arrival date/time,<br/>- proposed itinerary for new Freight Transport Execution Plan,<br/>- constraints on choice of route,<br/>- preferred modes to be used,<br/>- identity of the existing Freight Transport Execution Plan to be used as a starting point for the new Freight Transport Execution Plan, or null if none,<br/>- acceptance of proposal for the new Freight Transport Execution Plan, i.e. if not accepted prepare a revised Freight Transport Execution Plan.<br/>
