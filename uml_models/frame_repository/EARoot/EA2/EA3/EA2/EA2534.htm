It contains data objects that provide the text of a message that has been received from the Freight Vehicle Driver who is currently involved in the implementation of a Freight Transport Execution Plan and is to be output to the Freight Transport Operations Manager.  Extra data objects must be included to show in what form the message was received from the Driver, e.g. text, audio or both.<br/>
