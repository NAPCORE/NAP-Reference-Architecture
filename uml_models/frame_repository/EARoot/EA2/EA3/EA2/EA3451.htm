It contains data objects that provide data about a Vehicle whose presence in the road network will create an incident, e.g. road/winter maintenance vehicles, long/wide loads and vehicles that need to report themselves as travelling the wrong way through the road network.  These data objects will be used by the incident management functionality in the Manage Traffic Functional Area to create an "incident" that will be reported to the Driver.<br/>
