This Function shall be capable of providing the following facilities:<br/><br/>(1) The ability to provide facilities that enable an eCall message to be automatically sent to the Public Safety Answering Point (PSAP) by the Host Vehicle that has detected that it has been involved in an accident.<br/>(2) The ability to include the following data objects in the eCall message: VIN, current and previous Host Vehicle locations, Host Vehicle position can be trusted, direction in which the Host Vehicle is moving, fuel type, number of occupants, some basic information on the reason for the call, e.g. it is a real call, or a test call<br/>(3) The ability to check the current location of the Host Vehicle based on its own input from the Location Data Source.<br/>(4) The ability to send an eCall message to the PSAP either as a result of input from a Host Vehicle occupant, or automatically by the Driver monitoring system.<br/>(5) The ability for the Driver to cancel the automatic initiation of the eCall message.<br/>(6) The ability to display call acknowledgement messages from the Mobile Network functionality and the PSAP to the Driver and other occupants (if any) in the Host Vehicle.<br/>(7) The ability to re-send the eCall message if the Mobile Network functionality returns a "message failed" indication.<br/>

Details:
(a) when one of the accident detection data flows is received, the information on the reason for the call, and the current location of the vehicle is collected;<br/>(b) a request is made to the Operational Data Store management Function for data objects about the condition of the Driver;<br/>(c) check the location of the Host Vehicle provided in (a) above against the data objects received from the Location Data Source and if similar add a high confidence rating for the position of the Host Vehicle to the data.<br/>(d) check for any "cancellation" input data objects from the Driver's HMI.<br/>(e) a data flow containing the eCall message is sent to the eCall Management functionality;<br/>(f) a data flow is sent to the Vehicle Systems in the Host Vehicle via the Vehicle Communications functionality;<br/>(g) when the first reply is received from the PSAP, an acknowledgement that the call is being processed is output via the Driver's HMI in the eCall information data flow;<br/>(h) when the final reply is received from the PSAP, again a response is output via the driver's HMI in the eCall information data flow<br/>(i) if as a result of (e) a "message failed" response is received from the PSAP, repeat (e).<br/>
