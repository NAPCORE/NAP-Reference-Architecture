This Function shall be capable of providing the following facilities:<br/><br/>(1) Enable the initiation of the preparation of a new Freight Transport Execution Plan following a request received through other functionality in the Manage Freight and Fleet Operations Functional Area and using data objects provided by that functionality.<br/>(2) If required retrieve data objects containing an existing Freight Transport Execution Plan from the functionality that manages its storage.<br/>(3) Send new Freight Transport Execution Plans that have just been prepared to other functionality in the Manage Freight and Fleet Operations Functional Area for approval by the Freight Transport Operations Manager and Freight Transport User.<br/>(4) When a new Freight Transport Execution Plan is approved in (3) above send it to the functionality that manages its storage.<br/>(5) Upon request from the Freight Transport Operations Manager revise a Freight Transport Plan that is currently being implemented from the point in the Plan that has been reached.<br/>(6) Upon request from other functionality in the Manage Freight and Fleet Operations Functional Area revise a Freight Transport Plan that is currently being implemented from the point in the Plan that has been reached.<br/>

(a) continuously monitor for receipt of the prepare new FTEP data flow from the Provide FTOM Interface function;<br/>(b) when the data flow in (a) is received, check to see if an existing FTEP is specified and if so send it to the Manage FTEP Data Store function in the request existing FTEP details data flow;<br/>(c) as a result of (b) continuously monitor for receipt of the requested existing FTEP details data flow from the Manage FTEP Data Store function and when it arrives add its content to the data received in (b);<br/>(d) as a result of (b) or (c) send the collect data for FTEP preparation data flow to the Collect Data from FTOM and MMS function and continuously monitor for receipt of the prepared FTEP for use data flow from that function;<br/>(e) when the data flow in (d) is received send its content to the Provide FTOM Interface function in the prepared new FTEP data flow and continuously monitor for receipt of the save new FTEP to implement data flow;<br/>(f) when the data flow in (e) is received, send the prepared FTEP data to the Manage FTEP Data Store function in the new FTEP for store data flow;<br/>(g) continuously monitor for receipt of the prepare revised FTEP data flow and when it is received repeat (b) to (d) but replace the original origin for the FTEP with the current location of the Freight Vehicle provided in the data flow.<br/>
