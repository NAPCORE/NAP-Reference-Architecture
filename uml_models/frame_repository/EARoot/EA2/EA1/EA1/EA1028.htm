This High-level Function shall check that the user actually is allowed to perform the required transaction.  It shall check the access rights, the presence of the user on black lists, and if no incorrect payment (fraud) is detected, shall permit the transaction to take place and enable the user to continue to use the service.  The Function shall also record all the attempts to use a service without either funds or permission.<br/>
