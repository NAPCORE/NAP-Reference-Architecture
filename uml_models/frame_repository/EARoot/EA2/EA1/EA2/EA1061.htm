This Function shall be capable of providing the following facilities:<br/><br/>(1) The ability to receive from the Manage eCall Message Detection function details of each eCall message that has been sent from Vehicles, including the location of the Vehicle at the time the message was sent.<br/>(2) The ability to access the store of  Public Safety Answering Point (PSAP) Operational Data to find the PSAP that is the most appropriate to respond to the eCall message.<br/>(3) The ability to extract the details of how to communicate with the most appropriate PSAP and send them back to the Manage eCall Message Detection function.<br/>(4) The ability to provide contact information for all PSAP's that have that information stored on request from the Manage eCall Message Detection function.<br/>(5) The ability to update the content of the store of PSAP Operational Data whenever such input is received from the Provide PSAP Data Operator Interface.<br/>(6) The ability to extract contact details for the Vehicle from which an eCall message has been received and load them into the store of PSAP eCall Session Data (D2.4).<br/>(7) The ability to update D2.4 every time a notification is received from the Manage eCall Message Detection function that response is received from the PSAP to an eCall message and to delete all records of the Vehicle and the messages that it has received, when the response is finished. <br/>(8) The ability to provide details of the current content of the store of PSAP Operational Data whenever a request is received from the Provide PSAP Data Operator Interface.<br/>

(a) continuously monitor for receipt of a data flows from the Manage eCall Message Detection function and the Provide PSAP Data Operator Interface function;<br/>(b) when the data flow requesting PSAP contact details is received from the Manage eCall Message Detection function in (a), extract the location from it and access the content of the store of PSAP Operational Data;<br/>(c) use the content of the store of PSAP Operational Data to determine the most appropriate PSAP to which the eCall message shall be sent based on its location compared with that of the Vehicle that sent it, its status that was provided in its last response and period since it was sent the last eCall message;<br/>(d) as a result of (c) get the communication information for the PSAP from the store of PSAP Operation Data and send it to the Manage eCall Message Detection function;<br/>(e) update the log of eCall messages that have been sent to each PSAP for which details are contained in the store of PSAP Operational Data;<br/>(f) when the data flow received in (a) is from the Provide PSAP Data Operator Interface function, check its contents;<br/>(g) if the check in (f) shows that the data flow includes data objects containing new and/or updated PSAP details, load these into the store of PSAP Operational Data, in the case of updated details, overwriting what the store already contains;<br/>(h) if the check in (f) shows that the data flow includes a request for details of some or all the PSAP's in the store of PSAP Operational Data, extract the required information objects from the store;<br/>(i) when (h) has been completed, send the extracted information objects to the Provide PSAP Data Operator Interface function;<br/>(j) when the data flow containing the eCall message status is received from the Manage eCall Message Detection function in (a), update the content of the store of PSAP eCall Sessions Data adding details of new eCall messages, updating the details for eCall messages being answered and deleting eCall messages for which the responses have been completed;<br/>(k) when the data flow requesting contact information for all PSAP's is received from the Manage eCall Message Detection function in (a), extract the requested information and return it to the function;<br/>(l) when the data flow containing a list of available PSAP's is received from the Manage eCall Message Detection function in (a), update the store of PSAP Operational Data to show which PSAP's are available to receive eCall messages.<br/>
