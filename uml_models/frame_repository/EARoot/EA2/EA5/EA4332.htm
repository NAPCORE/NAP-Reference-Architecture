Appears In: DFD F7.3 Process Violation Notifications
This Data Store shall be used within the Provide Support for Law Enforcement Functional Area. It shall contain information objects that provide a record of all of the instructions sent to Vehicles that were acknowledged as having been displayed to Drivers.  The record of instructions will be used to determine whether or not a detected violation is genuine, or should be ignored because the Driver was only obeying instructions sent to the Vehicle.<br/><br/>The information objects in the Store shall contain the following data objects:<br/>(1) Date when record created (date string),<br/>(2) Time when record created (numbers defining hours, minutes and seconds),<br/>(3) Vehicle ID in which the instruction was displayed (number),<br/>(4) Type of instruction (characters),<br/>(5) Instruction details (characters and/or numbers),<br/>(6) Date at which instruction acknowledgement received (date string),<br/>(7) Time at which instruction acknowledgement received (numbers defining hours, minutes and seconds),<br/>(8) ID(s) road segment(s) to which the instruction applies (numbers).<br/><br/>It shall be possible for these data objects to be updated every time a new command is displayed to Drivers and for it to be removed after a suitable time period has elapsed since it was cancelled or replaced.<br/>
