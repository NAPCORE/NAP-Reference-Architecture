This Data Store shall be used within the Manage Public Transport Operations Functional Area.  It shall contain information objects that provide details of the fare schemes currently being operated for PT Journeys.<br/><br/>The information objects in the Store for each fare scheme shall contain the following data objects:<br/>(1) Fare scheme ID (number),<br/>(2) Fare scheme name (characters),<br/>(3) Date when fare scheme expires (date string),<br/>(4) Time(s) of day when fare scheme operates (numbers defining hours and minutes),<br/>(5) Route ID(s) to which fare scheme applies (numbers),<br/>(6) Type(s) of PT Passenger to which fare scheme applies (characters).<br/><br/>For each type of PT Passenger, the data objects shall be included to provide following fare information:<br/>(7) Range ID's of stop to which fare applies (numbers)<br/>(8) Numbers ID's of stops to which fare applies (numbers)<br/><br/>There shall be one information object for each fare scheme, which shall have a unique ID.  It shall be possible for there to be several fare schemes for the same route to cover different times of day, days of the week, months, years, etc.<br/>
