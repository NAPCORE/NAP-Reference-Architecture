Appears In: DFD F6.5.3 Trip Planning
This Data Store shall be used within the Provide Traveller Journey Assistance Functional Area (F6). It shall contain information objects that provide details about the road network and the traffic conditions within it for use in planning the Traveller's trips.<br/><br/>The information objects in the Store shall contain the following data objects:<br/>(1) Area covered (characters, e.g. nation, region, city, state, area names(s)),<br/>(2) Road network data (data for a digital roadmap using a standard format, e.g. GDF),<br/>(3) Link:<br/>    (3a) ID (number),<br/>    (3b) Current traffic flow (number in vehicles per hour),<br/>    (3c) Current average traffic speed (number in kph or mph),<br/>    (3d) Current average traffic headway (number in seconds),<br/>    (3e) Date for current traffic data (date string),<br/>    (3f) Time for current traffic data (numbers defining hours, minutes and seconds),<br/>    (3g) Predicted traffic flow (number in vehicles per hour),<br/>    (3h) Predicted average traffic speed (number in kph or mph),<br/>    (3i) Predicted average traffic headway (number in seconds),<br/>    (3j) Prediction date (date string),<br/>    (3k) Prediction time (numbers defining hours, minutes and seconds).<br/><br/>It shall be possible for link data object to be provided and stored for every link in the road network and for the current and predicted data objects to be updated whenever new data objects are received.  Also it shall be possible for there to be several sets of predicted data objects for different date/time combinations.<br/>
