This Terminator is part of the Driver Terminator and shall represent a human entity that drives a vehicle which is designed and licensed for the purpose of carrying freight of any kind.  It is expected that the Terminator shall provide inputs from which data objects can be created by system functionality to represent requests and data for use in the system and be capable of translating into human readable form data objects output from the functionality from within the system.  The Vehicle type for this Driver is defined in the description of the Freight Vehicle Terminator.  The functionality in the system must also be able to communicate with more than one human entity that is a Freight Vehicle Driver.<br/><br/>Mnemonic: d.fvd<br/>
