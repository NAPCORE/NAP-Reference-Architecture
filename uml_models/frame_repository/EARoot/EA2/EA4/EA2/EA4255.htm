This Terminator is part of the External Service Provider Terminator and shall represent an organisation that provides digital map data for use by functionality in the systems.  It shall be expected that this Terminator shall automatically send data objects containing digital map data when that data changes to functionality in various part of the system.  This Terminator shall also be expected to receive data objects containing requests for digital map data or details of the changes that need to be made to the existing digital map data.<br/><br/>Mnemonic: esp.g<br/>
