This is the representation of the FRAME functionality that appears as the central "box"  in the Context Diagram.  Clicking on the System "box" will open Data Flow Diagram 0 (DFD) which is the highest level of Data Flow Diagram.<br/>DFD0 contains all of the Functional Areas, the data flows that link them together and the data flows that link the Functional Areas to the Terminators in the Context Diagram.<br/>
