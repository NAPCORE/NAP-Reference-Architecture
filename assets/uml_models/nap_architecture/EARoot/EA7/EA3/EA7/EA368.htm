<b>Link to NAR: ???</b><br/><b><br/></b><b>“Data reuse – data provider”</b> KPI assesses, from the perspective of data provider, whether a data provider provides descriptive/detailed information about the terms and conditions for data reuse (or not). A value equal to 0 points out an absence of such a provision, a value equal to 1 indicates that there is a descriptive provision, while a value equal to 2 indicates that there is a detailed provision through sample contracts or standardized data licensing frameworks. It is assumed that a NAP should at least provide descriptive information about the terms and conditions for data reuse.<br/><br/>Is it possible for the NAP to provide “Terms and Conditions” for reuse of data published by the data provider at the NAP?<br/><br/>0: No<br/>1: Descriptive (brief text description)<br/>2: Detailed (where necessary – full sample contract conditions and/or standardized licenses framework) <br/><br/><br/>Minimum acceptable value for interoperability: 2<br/>
