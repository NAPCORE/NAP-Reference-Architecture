It contains outputs to the Data Provision System that have been created from data objects provided by functionality in the Manage Traffic Functional Area that are the response to a previous request to upload data. The response can be either “successful” or “failed”, which must be accompanied by a reason for the “failure” i.e. invalid authorisation credentials.  <br/>
