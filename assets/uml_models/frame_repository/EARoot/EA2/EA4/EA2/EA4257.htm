This Terminator is part of the External Service Provider Terminator and shall represent an organisation that provides travel information for non-road transport modes (rail, waterborne and air), including details of multi-modal exchange facilities.  It is expected that this Terminator shall be able to receive data objects containing requests from functionality in the system to provide travel information for non-road transport modes and for the requesting functionality to be able to receive and process the data objects the Terminator sends in reply.<br/><br/>Mnemonic: esp.mmtip<br/>
