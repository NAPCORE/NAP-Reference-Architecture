This Terminator shall represent different types of objects in the immediate geographic area that surrounds the Vehicle.  Each type of object shall be represented by one of the following Terminators:<br/><br/>Other Nearby Vehicles<br/>Other Road Users<br/>Stationary Object<br/><br/>It is expected that these Terminators shall provide data objects to the functionality in the system either directly through an interface or through sensors, and these shall be used by the functionality in the system to assist the Vehicle to navigate through the road network.  The functionality in the system shall identify the Vehicle in which is it operating as the Host Vehicle.<br/><br/>Mnemonic: ve<br/>
