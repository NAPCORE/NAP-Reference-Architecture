This Terminator is part of the Operator Terminator and shall represent the human entity that uses the functionality in the system to manage the way in which Vehicles are able to use one or more tunnels in the road network.  It is expected that the Terminator shall provide inputs from which data objects can be created by system functionality to represent commands, data and requests for use in the system and be capable of translating into human readable form data objects output from the functionality in the system.  The functionality in the system must also be able to communicate with more than one human entity that is a Tunnel Operator, each of which may belong to the same organisation, or to different organisations and may be responsible for one or more tunnels in the road network.<br/><br/>Mnemonic: o.tnlo<br/>
