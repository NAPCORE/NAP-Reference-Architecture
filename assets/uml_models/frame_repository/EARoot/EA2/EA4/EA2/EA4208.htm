This Terminator shall represent a diverse set of human entities that can perform privileged interactions with functionality in the system to manage the way it operates.  Each of the human entities shall be represented by one of the following Terminators:<br/><br/>Bridge Operator<br/>Emergency Operator<br/>Fleet Operator<br/>Freight Management Operator<br/>On-Demand Service Operator<br/>Parking Operator<br/>Public Transport Operator<br/>Road Maintenance Operator<br/>Road Network Operator<br/>Traveller Information Operator<br/>Tunnel Operator<br/>Un/Loading Zone Operator<br/><br/>It is expected that each Terminator shall provide inputs from which data objects can be generated by functionality in the system to represent requests and data for use by in the system.  Each Terminator is also expected to be capable of translating into human readable form data objects output from the functionality in the system and responding to that output if required.<br/><br/>Mnemonic: o<br/>
