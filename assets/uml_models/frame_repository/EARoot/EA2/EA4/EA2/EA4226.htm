This Terminator is part of the Driver Terminator and shall represent a human entity that as a Driver is able to plan their own routes using a trip planning service.  The Terminator shall provide inputs from which data objects can be created by system functionality to represent requests and information for use in the system and be capable of translating into human readable form data objects sent to the functionality from within the system.  This Terminator is used when the inputs/outputs from/to it do not apply to the Driver of a particular type of Vehicle.<br/><br/>Mnemonic: d.tpd<br/>
