It contains data objects provided by Traffic and Travel Information Provider Terminator which is part of the External Services Provider Terminator for use by functionality in the Provide Electronic Payment Facilities Functional Area.  The information objects contain all the elements necessary to determine the tariff for a given service and contract and include the following elements:<br/>- date,<br/>- service provider ID,<br/>- service ID,<br/>- for each service ID the tariff for every combination of parameters defining the service, and for various contract types.<br/><br/>
