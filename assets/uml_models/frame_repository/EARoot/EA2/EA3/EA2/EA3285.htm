It contains data objects that are used to indicate whether advanced payment for the service selected by the user has already been made.  The data flow contains the following data objects:<br/>- user ID and/or vehicle ID,<br/>- service ID,<br/>- service parameters needed to precisely qualify the selected service,<br/>- indicator of advanced payment (Yes / No),<br/>- percentage of fee already paid for,<br/>- date and location of advanced payment if any,<br/>- date and location of service utilisation.<br/>
