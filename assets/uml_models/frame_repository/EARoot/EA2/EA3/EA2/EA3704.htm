It contains data objects that provide all the information necessary to define the different types of contract that a user can place with an operator. The data flow includes the following data objects :<br/>- operator ID,<br/>- list of all potential contracts, for each one (some parameters being optional):<br/>   - service ID,<br/>   - other parameters defining the service (level of quality, etc.),<br/>   - geographical area covered,<br/>   - period covered,<br/>   - quantity of service purchased,<br/>   - tariff,<br/>   - mode of payment.<br/>
