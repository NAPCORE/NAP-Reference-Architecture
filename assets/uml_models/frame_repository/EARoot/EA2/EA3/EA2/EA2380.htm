It consists of input from the Pre-Trip Traveller from which data objects are created that provide basic data about the trip that is to be planned and are for use by functionality in the Provide Traveller Journey Assistance Functional Area.  The basic data may include details about the trip such as, the date on which the trip will be made (or start), the locations of the origin and destination of the trip, the departure and arrival times, places to be visited (or passed through, i.e. way points) for the trip, places to be avoided, travel modes to be used and/or avoided, the type of road vehicle that will be used for all of part of the trip, the identity of the Traveller preparing the trip (enables their General Trip Preferences (GTP) to be used, if available), the number of Travellers making the trip and information about any goods that are being carried.<br/>
