It contains output to the Heavy Goods Vehicle Driver that have been created from data objects provided by functionality in the Provide Support for Co-operative Systems Functional Area and includes an indication of the success or failure of a previously requested route creation.  If the creation has been successful, data objects containing the unique identifier for the route will be included, but if not then data objects will be included to indicate why the creation failed.<br/>
