It contains inputs from the Pedestrian from which data objects are created that enable functionality in the Provide Support for Host Vehicle Systems Functional Area to determine the physical presence of a pedestrian in the vicinity of the host vehicle and track its movements.  The definition of a "pedestrian" can include walkers, those in wheelchairs, those on horses, and any other form of Traveller that is not using a type of wheeled vehicle to move from one place to another.<br/>
