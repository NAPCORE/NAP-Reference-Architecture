It contains data objects that define the movement of funds to be performed between the EP account of the user, and the account of the operators.  The data flow contains the following data objects:<br/>- date / time,<br/>- user ID,<br/>- service ID,<br/>- list of operators or information providers ID's,<br/>- for each operator (or information provider),<br/>       account ID,<br/>       amount to be transferred.<br/>
