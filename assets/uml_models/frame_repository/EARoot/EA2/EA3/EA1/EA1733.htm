It contains inputs from the Other Vehicle from which data objects are created that provide all the information that will be used by functionality in the Provide Support for Host Vehicle Systems Functional Area.  The identity and location of the Other Vehicle will be included so that the Host Vehicle can determine the relevance of what the Other Vehicle has provided.<br/>
