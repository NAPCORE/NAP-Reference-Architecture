Appears In: DFD F4.5 Provide PT Fare Facilities
This Data Store shall be used within the Manage Public Transport Operations Functional Area.  It shall contain information objects that provide details of the fare credit that is currently available for use by the PT Passenger.<br/><br/>The information objects in the Store for each PT Passenger's fare credit shall contain the following data objects:<br/>(1) Amount of credit (number)<br/>(2) Date of last top up (date string)<br/>(3) Date of expiry (date string)<br/><br/>Each information object in this Data Store shall represent a PT "fare card" in its simplest form and is intended to be implemented that it forms part of the physical objects that go with each PT Passenger.  It is expected that either there will not be any personal identification data relating to the "owning" PT Passenger, or such data will be provided by mechanisms that are outside the scope of the functionality provided by the FRAME Architecture.<br/>
