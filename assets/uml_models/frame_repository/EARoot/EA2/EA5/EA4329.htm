Appears In: DFD F7.5 Manage Rules and Users' Registrations
This Data Store shall be used within the Provide Support for Law Enforcement Functional Area.  It shall contain an information objects that provide all the rules and the types of violation associated with each rule.<br/><br/>Each information object in the Store shall be for one rule and contain the following data objects:<br/>(1) Rule ID (number),<br/>(2) Area of road network to which the rule applies (characters),<br/>(3) Description of the rule (characters),<br/>(4) Types of violation that can be associated with the rule (characters).<br/><br/>The data objects in the Store f or each type of violation shall be structured in the following way:<br/>(5) Violation ID (number),<br/>(6) Violation description (characters),<br/>(7) Level of seriousness (characters),<br/>(8) Authority responsible for enforcement (characters),<br/>(9) Enforcement procedure (characters),<br/>(10) Potential sanction (characters).<br/><br/>It shall be possible for these data objects to be updated every time a change is made.<br/>
