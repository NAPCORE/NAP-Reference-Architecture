Appears In: DFD F7.5 Manage Rules and Users' Registrations
This Data Store shall be used within the Provide Support for Law Enforcement Functional Area.  It shall contain an information object that provides details of registered vehicles and users.<br/><br/>The information objects in the Store shall contain the following data objects:<br/>(1) Vehicle ID (number),<br/>(2) Vehicle type (characters),<br/>(3) Vehicle owner ID (number),<br/>(4) Vehicle driver ID (number),<br/>(5) Vehicle operator ID (number),<br/>(6) Maximum authorised weight (number in kg or lbs),<br/>(7) Maximum authorised speed (number in kph or mph),<br/>(8) Vehicle user ID (number),<br/>(9) ID of any other vehicles used by the user (number),<br/>(10) Operations allowed for the vehicle (characters),<br/>(11) Start date for vehicle registration (date string),<br/>(12) End date for vehicle registration (date string),<br/>(13) Level of pollution produced by the vehicle (numbers as percentages for each pollutant).<br/><br/>It shall be possible for the Vehicle owner, Driver, operator and user to be the same or different people and/or organisations.<br/>
