This Data Store shall contain information objects that define each Freight Transport Execution Plan (FTEP) that is capable of being implemented by functionality in the system.  Each Freight Transport Execution Plan shall consist of one information object which shall comprise, but not be limited to, the following data objects:<br/>(1) FTEP identity (number),<br/>(2) date FTEP was originally created (number),<br/>(3) time FTEP was originally created (number),<br/>(4) number of modifications since first creation (number),<br/>(5) for each modification:<br/>       (a) date FTEP was modified (number),<br/>       (b) time FTEP was modified (number),<br/>       (c) description of modification (characters),<br/>(6) number of Freight Transport Load Items (FTLI's) to be moved by this FTEP (number),<br/>(7) for each FTLI:<br/>       (a) location of origin (number),<br/>       (b) date to be collected (number),<br/>       (c) time to be collected (number),<br/>       (d) description of FTLI, e.g. liquid in its own container, liquid requiring a container, powder and hazardous designation (characters),<br/>       (e) size of FTLI, l, w, h (numbers),<br/>       (f) is FTLI to be considered an abnormal load (y/n),<br/>       (g) location of destination (number),<br/>       (h) expected date to be delivered to destination (number),<br/>       (i) expected time to be delivered to destination (number),<br/>       (j) location(s) of way point(s) on road based route to be followed (numbers),<br/>       (k) other modes of transport to be used - blank if not required (characters),<br/>       (l) location of Freight Terminal Area (FTA)  at which FTLI must be delivered from which it will be taken forward by other mode of transport (number),<br/>       (m) expected date of delivery to FTA (number),<br/>       (n) expected time of  delivery FTA (number),<br/>       (o) location of FTA from which FTLI must be collected having been delivered by other mode of transport (number),<br/>       (p) expected date of collection from FTA (number),<br/>       (q) expected time of collection from FTA (number),<br/>       (r) location(s) of way point(s) on road based route to be followed from FTA to destination (numbers)<br/>       (s) any other special requirements (characters).<br/>(8) number of Freight Vehicles required to carry the FTLI(s) (number),<br/>(9) type(s) of Freight Vehicle required to carry the FTLI(s) (characters),<br/>(10) details of any relevant traffic management policies that apply to this FTEP (characters).<br/>
