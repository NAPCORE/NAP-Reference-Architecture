This Data Store shall be used within the Provide Support for Host Vehicle Services Functional Area.  It shall contain three information objects covering the Vehicle, the Driver and the traffic conditions.<br/><br/>The information objects in the Store about the Vehicle shall contain the following data objects:<br/>(1) Vehicle ID (number - could be the VIN),<br/>(2) Type (characters),<br/>(3) Date (date string),<br/>(4) Time (numbers defining hours, minutes and seconds),<br/>(5) Location (characters and/or numbers, e.g. GPS/Galileo data),<br/>(6) Speed (number in kph or mph),<br/>(7) Acceleration (number),<br/>(8) Brakes applied (number or character indicating YES or NO),<br/>(9) Windscreen wipers on (number or character indicating YES or NO),<br/>(10) Lights on (number or character indicating YES or NO),<br/>(11) Outside temperature (number or character indicating YES or NO),<br/>(12) ABS operating (number or character indicating YES or NO),<br/>(13) ESP operating (number or character indicating YES or NO),<br/>(14) Driver command (character),<br/>(15) Date driver command input (date string),<br/>(16) Time driver command input (numbers defining hours, minutes and seconds as a minimum).<br/><br/>The information objects in the Store about the Vehicle shall contain the following data objects:<br/>(17) Current status (characters),<br/>(18) Alcohol detected (number or character indicating YES or NO).<br/><br/>The information objects in the Store about the Vehicle shall contain the following data objects:<br/>(19) Area in which Vehicle is operating (characters, e.g. nation, region, city, state, area names(s)),<br/>(20) Road network data (data for a digital roadmap using a standard format, e.g. GDF),<br/>(21) Traffic conditions:<br/>    (21a) Flow (number in vehicles per hour),<br/>    (21b) Speed (number in kph or mph),<br/>    (21c) Headway (number in seconds).<br/><br/>It shall be possible for the above data objects to be updated in real-time as the Vehicle moves through the road network.<br/>
