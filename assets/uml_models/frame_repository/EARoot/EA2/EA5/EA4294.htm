Appears In: DFD F2.1.2 Manage Emergency Intervention
This Data Store shall be used within the Provide Security and Emergency Facilities Functional Area.  It shall contain information objects that provide details of all incident/alarm notifications (including mayday calls) that have been received by the functionality in this Area.  It is in two parts; un-processed and processed emergencies.<br/><br/>Each information object in the Store for un-processed emergencies shall contain the following data objects:<br/>(1) Time (numbers defining hours, minutes and seconds),<br/>(2) Date (date string),<br/>(3) Incident location (characters and/or numbers, e.g. GPS/Galileo data),<br/>(4) Type of vehicle involved in the incident (characters),<br/>(5) Known status of each vehicle (characters),<br/>(6) Description of cargo (if any) carried by each vehicle (characters),<br/>(7) The number of people involved in the incident (number),<br/>(8) Health status of each person involved in the incident (characters),<br/>(() Any additional information relevant for emergency process (characters),<br/>(10) Definition of originator of the incident notification (characters),<br/>(11) System that was the source of the incident notification (characters.<br/><br/>The data for processed emergencies held in the Store shall be structured in the following way:<br/>(12) A consolidated problem description comprising:<br/>   (12a) Time (numbers defining hours, minutes and seconds),<br/>   (12b) Location (characters and/or numbers, e.g. GPS/Galileo data),<br/>   (12c) Type of vehicle involved in the incident (characters),<br/>   (12d) Known status of each vehicle (characters),<br/>   (12e) Description of cargo (if any) carried by each vehicle (characters),<br/>   (12f) The number of people involved in the incident (number),<br/>   (12g) Health status of each person involved in the incident (characters),<br/>   (12h) Any additional information relevant for emergency process (characters),<br/>   (12i) A list of all associated incidents (characters),<br/>(13) Description of planned actions comprising:<br/>     (13a) The emergency services that will be involved in the action, (characters),<br/>     (13b) Time when action will start (numbers defining hours, minutes and seconds),<br/>     (13c) The number of vehicles involved (number),<br/>     (13d) A description of the result of each action (characters),<br/>     (13e) A list of progress reports for each action (characters).<br/><br/>It shall be possible for there to be several vehicles involved in each incident (un-processed or processed), each with their own cargos and/or numbers of occupants.  Similarly it shall be possible for the number of people involved in the incident to be greater than the number of vehicle occupants if people outside the vehicles are involved in the incident.  It shall be possible for a vehicle to include a bicycle or a powered two wheeled (PTW) vehicle.<br/>
