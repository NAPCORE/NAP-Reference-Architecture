Appears In: DFD F5.15.1 Provide Detection for the Vehicle Surroundings, Functional Tree of Area F5 - High-level Function F5.15 only
This Function shall be capable of providing the following facilities:<br/><br/>(1) The ability to take responsibility for processing information from sensors that detect the presence of Pedestrians near to the Host Vehicle, i.e. the Vehicle in which it is operating.<br/>(2) The ability to process the sensor data to determine the location of the Pedestrians (including Vulnerable Road Users (VRU's)) relative to the Host Vehicle and their probable trajectory.<br/>(3) Once processing is complete, the ability to send the data in digital form to the Determine and store local Host Vehicle Conditions Function.<br/>
