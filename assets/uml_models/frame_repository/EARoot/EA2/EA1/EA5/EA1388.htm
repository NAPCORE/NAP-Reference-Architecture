Appears In: DFD F5. Provide Support for Host Vehicle Services, Functional Tree of Area F5 - High-level Function F5.15 only, Functional Tree of Area F5 Provide Support for Host Vehicle Services
This High-level Function shall provide facilities that enable the Host Vehicle to detect various forms of entity that are in its immediate surroundings.  These forms of entity shall include equipped Vehicles, other road users (i.e. non-equipped Vehicles, Cyclists and Motorcyclists), Pedestrians, Atmospheric Environment, visibility, the road surface and Stationary Objects.  The functionality shall create data objects to represent these entities and combine them with data objects it collects about the way that the Host Vehicle is operating to determine any possible conflicts and/or unsafe situations.  The Driver of the Host Vehicle will be sent warnings about these by the functionality, which will also pass the data objects about the conflict and/or unsafe situations to Other Vehicles.  Some of the data objects collected, such as visibility, will be combined with the data provided by Other Vehicles to provide the Driver of the Host Vehicle with a more comprehensive picture of the area around it.<br/>
