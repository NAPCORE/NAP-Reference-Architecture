This Function shall be capable of providing the following facilities:<br/><br/>(1) The ability to receive road network static data from both urban and inter-urban functionality<br/>(2) The ability to process the received data so that it can be used in the road network model by the Traffic Simulation Engine functionality.<br/>(3) When the data has been processed it shall be sent to the functionality that manages the store of Traffic Simulation Data from where it can be obtained by the Traffic Simulation Engine functionality.<br/>
