This Function shall be capable of providing the following facilities:<br/><br/>(1) A HMI through which the PT Operator can manage the operation of the PT Fleet.<br/>(2) The HMI shall enable the PT Operator to set the criteria which is used to decide if and when PT Vehicle or service priority is requested and to request the introduction of additional PT Vehicles.<br/>(3) The HMI shall enable the PT Operator to confirm additional PT Vehicles requested by other modes and to request report on the PT Control actions that have been taken.<br/>(4) The HMI shall enable the PT Operator to provide data about each PT Vehicle in the fleet so that real-time and historic data can be properly associated with each Vehicle.<br/>

Details:
(a) continuously monitor for the receipt of the change service delivery criteria, request actions report data flows and PT vehicle static data from the PT operator;<br/>(b) when the first data flow in (a) is received send the changed criteria to the Optimise PT Fleet Control function in the updated service delivery criteria data flow;<br/>(c) when the second data flow in (a) is received send the request actions report data flow to the Optimise PT Fleet Control function;<br/>(d) as a result of (c) continuously monitor for the receipt of the requested action report data flow;<br/>(e) when the data flow in (d) is received, send its contents to the PT operator in the actions report data flow;<br/>(f) continuously monitor for the receipt of the confirm PT service change data flow and when it is received send the request for service change data flow to the PT operator;<br/>(g) when as a result of (f) the service change response data flow is received from the PT operator, out the result in the confirmed PT service change data flow to the Optimise PT Fleet Control function;<br/>(h) when the third data flow in (a) is received, send the data that it contains to the store of Real Time PT Vehicle Status and the store of Historical PT Vehicle Data in the PT vehicle static data for real time use and PT vehicle static data for historic use data flows respectively.<br/>
