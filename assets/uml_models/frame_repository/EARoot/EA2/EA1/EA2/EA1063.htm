Appears In: Functional Tree of Area F2 Provide Safety and Emergency Facilities, DFD F2.1 Manage Emergencies
This Function shall be capable of providing the following facilities:<br/><br/>(1) The maintenance of a Data Store containing data that is used during the processing of an incident or an emergency.<br/>(2) The ability to send requests for updates (refresh) of the route previously provided for an Emergency Vehicle based on new data, or input from the Emergency Operator received from the Operator Interface Function.<br/>(3) The provision of data to the Emergency Operator following a specific request made through the Operator Interface Function.<br/>(4) The ability to receive data about an incident or emergency from the Emergency Services.<br/>(5) The ability to receive updated map data for use in the response to the incident or emergency.<br/>(6) The provision of common data for use in creating emergency intervention plans and classifying emergencies that has been.<br/>
